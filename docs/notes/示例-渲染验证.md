## 代码高亮

```cpp
#include <iostream>
using namespace std;

int main() {
    // 快读快写示例
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    long long sum = 0;
    for (int i = 1; i <= n; ++i) sum += i;
    cout << sum << '\n';
    return 0;
}
```

支持行高亮：

```python{2,4}
def fib(n):
    if n < 2:        # 高亮
        return n
    return fib(n-1) + fib(n-2)  # 高亮
```

## 数学公式

行内公式：当 $a \ne 0$ 时，方程 $ax^2 + bx + c = 0$ 的解为 $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$。

块级公式：

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

$$
\begin{aligned}
f(x) &= \int_{-\infty}^{\infty} \hat{f}(\xi)\, e^{2\pi i \xi x} \, d\xi \\
     &= \mathcal{F}^{-1}\{\hat{f}\}(x)
\end{aligned}
$$
