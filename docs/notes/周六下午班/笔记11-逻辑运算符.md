# 笔记 11 · 逻辑运算符

本节课主要学习了C++中的三个逻辑运算符：

1. **非（!）**：逻辑取反。真的变成假的，假的变成真的。

2. **与（&&）**：逻辑与（并且）。要求两边的条件同时成立，结果才为真。

3. **或（||）**：逻辑或（或者）。只要两边有一个条件成立，结果就为真。

**特别注意：** 当“与（&&）”和“或（||）”在同一个条件判断中同时使用时，因为“与”的优先级高于“或”，为了避免逻辑混乱和错误，**强烈建议使用括号 `()` 来明确计算的先后顺序**，这样既能保证正确性，又能让代码更容易阅读。

## [P129. 【例22.1】 有一门课不及格的学生](https://kuaima.nz/p/P129)

**题目说明：** 给出一名学生的语文和数学成绩，判断他是否恰好有一门课不及格(成绩小于60分)。若该生恰好有一门课不及格，输出1；否则输出0。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a, b; // 定义两个整数变量a和b，分别用来表示语文和数学成绩
    cin >> a >> b;

    if ((a < 60 && b >= 60) || (a >= 60 && b < 60)) // 判断是否恰好只有一门课不及格，使用括号明确“与”和“或”的优先级
    {
        cout << 1 << endl;
    }
    else // 如果两门都不及格，或者两门都及格
    {
        cout << 0 << endl;
    }

    return 0;
}
```

## [P119. 练20.4 判断点在第几象限](https://kuaima.nz/p/P119)

**题目说明：** 输入2个整数x、y值，表示平面上一个坐标点，判断该坐标点处于第几象限，并输出相应的结果。用数字1, 2, 3, 4分别对应四个象限。

![四象限示意图](/images/quadrant.png)

我们可以直观地把平面分成四个部分（象限）。通过观察横轴（x轴）和纵轴（y轴）的正负号，就能轻松判断点的位置：

- **第一象限（右上）**：横坐标为正，纵坐标为正 ➡️ `x > 0 并且 y > 0`

- **第二象限（左上）**：横坐标为负，纵坐标为正 ➡️ `x < 0 并且 y > 0`

- **第三象限（左下）**：横坐标为负，纵坐标为负 ➡️ `x < 0 并且 y < 0`

- **第四象限（右下）**：横坐标为正，纵坐标为负 ➡️ `x > 0 并且 y < 0`

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x, y; // 定义两个整数变量x和y，用来表示平面上的坐标点
    cin >> x >> y;

    if (x > 0 && y > 0) // 第一象限特征
    {
        cout << 1 << endl;
    }
    else if (x < 0 && y > 0) // 第二象限特征
    {
        cout << 2 << endl;
    }
    else if (x < 0 && y < 0) // 第三象限特征
    {
        cout << 3 << endl;
    }
    else if (x > 0 && y < 0) // 第四象限特征
    {
        cout << 4 << endl;
    }

    return 0;
}
```

## [P120. 练20.5 点和正方形的关系](https://kuaima.nz/p/P120)

**题目说明：** 有一个正方形，四个角的坐标分别是 (1, -1), (1, 1), (-1, -1), (-1, 1)。判断一个给定的点是否在这个正方形内（包括正方形边界）。如果点在正方形内，则输出"yes"，否则输出 "no"。

要判断一个点是不是在正方形内，不要把 x 和 y 混在一起想。我们可以**把横坐标 x 和纵坐标 y 拆开来，独立进行说明**：

1. **单独看左右方向（x轴）**：正方形的最左边是 -1，最右边是 1。所以要想在正方形内，横坐标必须满足：`x >= -1 并且 x <= 1`。

2. **单独看上下方向（y轴）**：正方形的最下边是 -1，最上边是 1。所以要想在正方形内，纵坐标必须满足：`y >= -1 并且 y <= 1`。只有当 `x` 满足它的条件，**并且** `y` 也满足它的条件时，这个点才真正被框在这个正方形内部。

**代码实现（常规版）：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x, y; // 定义两个整数变量x和y，用来存储需要判断的点的坐标
    cin >> x >> y;

    // 利用逻辑与（&&）将x的独立条件和y的独立条件连接起来
    if (x >= -1 && x <= 1 && y >= -1 && y <= 1)
    {
        cout << "yes" << endl;
    }
    else // 如果有任何一个条件不满足（比如跑出了左边界或者跑出了上边界）
    {
        cout << "no" << endl;
    }

    return 0;
}
```

除了分别判断上下左右边界，我们还可以借助数学中的绝对值概念。点 `(x, y)` 到 y轴的距离是 $|x|$，到 x轴的距离是 $|y|$。由于这个正方形的中心正好在原点 $(0, 0)$，且边长为 2（向上下左右各延伸 1 个单位），因此只要满足 $|x| \le 1$ 并且 $|y| \le 1$，点就在正方形内。在C++中，我们可以使用 `abs()` 函数来求绝对值。

**代码实现（绝对值版）：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x, y;
    cin >> x >> y;

    // 使用abs()函数求绝对值，代码逻辑更加简洁
    if (abs(x) <= 1 && abs(y) <= 1)
    {
        cout << "yes" << endl;
    }
    else
    {
        cout << "no" << endl;
    }

    return 0;
}
```

> **拓展小知识：** 这种利用坐标差的绝对值的最大值来衡量距离的方式，实际上被称为 **切比雪夫距离（Chebyshev distance）**。感兴趣的同学，课后可以自行去了解一下计算机和数学中其他常见的距离表示方法，比如 **曼哈顿距离** 和 **欧氏距离** 哦！
