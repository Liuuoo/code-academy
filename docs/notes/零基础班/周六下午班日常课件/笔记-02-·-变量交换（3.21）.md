# 笔记 02 · 变量交换（3.21）


## 1. 阿布拉2

**题目链接**： https://kuaima.nz/p/P19

**解题思路**： 通过引入一个第三方临时变量（`temp`）来暂存其中一个变量的值，从而实现两个变量的值互换。

**参考代码**：

```cpp
#include <bits/stdc++.h> // 引入C++万能头文件，包含了几乎所有的标准库
using namespace std; // 使用标准命名空间，方便直接调用cin和cout等

int main() // 定义主函数，这是C++程序的入口点
{
    int a, b; // 声明两个整型变量a和b，用于存放输入的墨水数量
    cin >> a >> b; // 从标准输入流（键盘）读取两个整数，并分别赋值给a和b
    
    // 输出交换前的状态
    cout << "a=" << a << " b=" << b << endl; // 在屏幕上输出"a="、变量a的值、空格和"b="、变量b的值，并换行
    
    // 借助第三方变量进行交换
    int temp = a; // 声明一个整型临时变量temp，并将a当前的值赋给它
    a = b; // 将变量b的值赋给变量a
    b = temp; // 将temp中暂存的（原先a的）值赋给变量b，完成交换过程
    
    // 输出交换后的状态
    cout << "a=" << a << " b=" << b << endl; // 在屏幕上输出交换后的变量状态，并换行
    
    return 0; // 程序正常执行完毕，向操作系统返回状态码0
} // 主函数作用域结束
```

## 2. 计算电影票总价

**题目链接**： https://kuaima.nz/p/P20

**解题思路**： 已知单价为10元，总价 = 人数 × 10。注意输出格式要求人数和总价之间用一个空格隔开。

**参考代码**：

```cpp
#include <bits/stdc++.h> // 引入C++万能头文件
using namespace std; // 声明使用std标准命名空间

int main() // 程序的执行入口，主函数
{
    int x; // 声明一个整型变量x，用来表示小朋友的人数
    cin >> x; // 从标准输入读取小朋友的人数，存入变量x
    
    // 输出人数，加一个空格，再输出总价
    cout << x << " " << x * 10 << endl; // 依次输出人数x、一个空格字符串、计算得到的总价(x*10)，最后换行
    
    return 0; // 退出主函数，返回0表示程序执行成功
} // 结束主函数的代码块
```

## 3. CSP复赛总成绩

**题目链接**： https://kuaima.nz/p/P21

**解题思路**： 依次读取四道题的得分，并将它们累加起来输出即可。

**参考代码**：

```cpp
#include <bits/stdc++.h> // 引入包含了众多标准库的万能头文件

{

```

## 4. 捡石头

**题目链接**： https://kuaima.nz/p/P22

**解题思路**： 四块石头总重量固定为20斤，已知前三块的重量。第四块的重量 = 20 - 第1块 - 第2块 - 第3块。

**参考代码**：

```cpp
#include <bits/stdc++.h> // 包含所有的标准头文件
using namespace std; // 使用标准空间名称std

int main() // 主函数启动
{
    int a, b, c; // 声明三个整型变量a, b, c，用来保存前三块石头的重量
    cin >> a >> b >> c; // 读取用户输入的前三块石头的重量数值
    
    // 用总重减去前三块的重量
    cout << 20 - a - b - c << endl; // 总重量20减去已知的三个重量，计算出第四块石头的重量，输出并换行
    
    return 0; // 返回整型值0，程序正确执行结束
} // 结束主函数
```

## 5. 套盒子

**题目链接**： https://kuaima.nz/p/P23

**解题思路**： 使用乘法原理计算总数：大盒子数量为 a，中盒子数量为 `a * b`，小盒子数量为 `a * b * c`。总数量为这三者之和。

**参考代码**：

```cpp
#include <bits/stdc++.h> // 使用万能头文件，简化头文件的编写
using namespace std; // 引入std以直接使用标准库的函数和对象

int main() // 定义主入口函数
{
    int a, b, c; // 声明三个整型变量，分别存储不同层级盒子的倍数或数量基准
    cin >> a >> b >> c; // 读取输入的三个整数a、b和c
    
    // 分别计算三种盒子的数量并求和
    int total = a + (a * b) + (a * b * c); // 按照公式：大盒子数量 + 中盒子数量 + 小盒子数量，计算出盒子总数并赋给total变量
    cout << total << endl; // 将盒子总数total的值输出至屏幕，之后进行换行操作
    
    return 0; // 正常退出程序
} // main函数结束
```

## 7. 兴趣小组 (容斥原理)

**题目链接**： https://kuaima.nz/p/P25

**解题思路**： 利用容斥原理： 参加了至少一个小组的人数 = 语文组人数(a) + 数学组人数(b) - 两个组都参加的人数(c)。 两个组都没参加的人数 = 全班总人数(n) - 参加了至少一个小组的人数。

**参考代码**：

```cpp
#include <bits/stdc++.h> // 引入可以代替绝大多数头文件的万能库
using namespace std; // 声明命名空间

int main() // 主程序逻辑在这里编写
{
    int n, a, b, c; // 声明四个变量：n代表总人数，a为参加语文人数，b为参加数学人数，c为两项都参加人数
    cin >> n >> a >> b >> c; // 依次获取输入流当中的四个相应的整数数据
    
    // 计算至少参加了一个小组的人数
    int joined = a + b - c; // 根据容斥原理公式计算出参与了兴趣小组的总人数，并将其储存在joined中
    // 计算什么都没参加的人数
    int none = n - joined; // 利用全班总人数减去参与了兴趣小组的人数，得到未参加任何小组的人数，存放在none中
    
    cout << none << endl; // 将未参加任何小组的人数结果输出，并且输出换行符
    
    return 0; // 完成任务，成功退出程序
} // 函数域结束
```

## 8. 求3个整数的和

**题目链接**： https://kuaima.nz/p/P26

```cpp

int main() // C++程序的必备主函数

    cin >> a >> b >> c; // 从键盘接收这三个整数的输入
    
    // 输出三个数的和
```
