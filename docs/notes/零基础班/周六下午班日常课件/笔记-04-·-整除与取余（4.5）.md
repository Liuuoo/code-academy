# 笔记 04 · 整除与取余（4.5）


## 课堂知识点回顾：整除与取余

在本次课程中，我们重点学习了C++中的两个重要数学运算符：**整除（`/`）** 和 **取余（`%`）**。

**直观理解：** 我们可以用的“带余除法”来理解这两个符号的工作原理。
举个例子：数学中我们算 **15 ÷ 7 = 2 …… 1** （商是2，余数是1）。 在C++中，这两个结果需要用不同的符号单独获取：

- 使用 **整除（`/`）**：`15 / 7` 得到的就是商，结果为 **2**。
    
- 使用 **取余（`%`）**：`15 % 7` 得到的就是余数，结果为 **1**。
    

1. **整除（`/`）—— 拿到整数部分（商）**
    
    - 当两个整数相除时，C++会丢弃小数部分，只保留整数部分。
        
    - **特殊用法**：
        
        - `变量 / 10`：可以把一个数字的**最后一位去掉**。例如 `123 / 10` 结果是 `12`。
            
        - `变量 / 100`：可以把一个数字的**最后两位去掉**。例如 `1234 / 100` 结果是 `12`。
            
2. **取余（`%`）—— 拿到余数部分**
    
    - 取余运算也叫模运算，用于计算两个整数相除后剩下的余数。
        
    - **特殊用法**：
        
        - `变量 % 10`：可以拿到一个数字的**个位（最后一位）数字**。例如 `123 % 10` 结果是 `3`。
            
        - `变量 % 100`：可以拿到一个数字的**最后两位数字**。例如 `1234 % 100` 结果是 `34`。
            

## [P33. 【例8.1】人民币支付](https://www.google.com/search?q=https://kuaima.nz/p/P33 "null")

**题目说明：** 给定一个金额（以元为单位），要求计算支付该金额时，100元、50元、20元、10元、5元、1元各需要多少张，要求尽量使用大面额的钞票。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 定义一个整数变量n，用来存储需要支付的总金额
    cin >> n; // 从输入中读取总金额
    cout << n / 100 << endl; // 计算并输出需要多少张100元（整除100），然后换行
    n = n % 100; // 更新剩余金额：用取余操作拿掉100元的部分，留下剩下的钱
    cout << n / 50 << endl; // 计算并输出需要多少张50元，然后换行
    n = n % 50; // 更新剩余金额：用取余操作拿掉50元的部分
    cout << n / 20 << endl; // 计算并输出需要多少张20元，然后换行
    n = n % 20; // 更新剩余金额：用取余操作拿掉20元的部分
    cout << n / 10 << endl; // 计算并输出需要多少张10元，然后换行
    n = n % 10; // 更新剩余金额：用取余操作拿掉10元的部分
    cout << n / 5 << endl; // 计算并输出需要多少张5元，然后换行
    n = n % 5; // 更新剩余金额：用取余操作拿掉5元的部分
    cout << n << endl; // 剩下的金额全都是1元，直接输出并换行
    
    return 0;
}
```

## [P34. 【例8.2】成绩](https://www.google.com/search?q=https://kuaima.nz/p/P34 "null")

**题目说明：** 已知一门课程的总成绩 = 作业成绩×20% + 小测成绩×30% + 期末考试成绩×50%。给出这三项的成绩，计算出最终的总成绩。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a, b, c; // 定义三个整数变量a, b, c，分别代表作业成绩、小测成绩和期末成绩
    cin >> a >> b >> c; // 从输入中一次性读取这三项成绩
    int total = a * 20 / 100 + b * 30 / 100 + c * 50 / 100; // 按照百分比公式计算各项得分并相加，求出总成绩
    cout << total << endl; // 输出最终计算得到的总成绩并换行
    
    return 0;
}
```

## [P35. 【例8.3】计算球的体积](https://www.google.com/search?q=https://kuaima.nz/p/P35 "null")

**题目说明：** 给定一个球的半径 $r$，根据公式 $V = \frac{4}{3} \times \pi \times r^3$ 计算球的体积，其中 $\pi$ 取 3.14。结果保留2位小数输出。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double r; // 定义一个双精度浮点型变量r，用来存储球的半径
    cin >> r; // 从输入中读取球的半径数值
    double v = 4.0 / 3.0 * 3.14 * r * r * r; // 根据公式计算体积，注意使用4.0/3.0确保除法是小数运算而不是整数运算
    cout << fixed << setprecision(2) << v << endl; // 使用保留小数的格式，将体积v保留2位小数输出并换行
    
    return 0; 
}
```

## [P36. 练8.1 尼克与强盗](https://www.google.com/search?q=https://kuaima.nz/p/P36 "null")

**题目说明：** 输入一个两位数的门牌号，将这个两位数的十位数字与个位数字调换位置，然后输出新的门牌号。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 定义一个整数变量n，用来存储原本的两位数门牌号
    cin >> n; // 从输入读取这个两位数
    int shi = n / 10; // 利用"整除10"的技巧，获取该数字的十位数字
    int ge = n % 10; // 利用"取余10"的技巧，获取该数字的个位数字
    cout << ge << shi; // 利用输出不换行/空格直接“拼出”交换的数
    return 0;
}
```
