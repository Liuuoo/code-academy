## [P33. 【例8.1】人民币支付](https://kuaima.nz/p/P33 "null")

**题目说明：** 给定一个金额（以元为单位），现在想知道支付该金额的各种面额的人民币数量，显示100元，50元，20元，10元，5元，1元各多少张，要求尽量使用大面额的钞票。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 定义整型变量n，表示要支付的总金额
    cin >> n; // 读取用户输入的金额数字
    cout << n / 100 << endl; // 计算100元面额的张数并输出
    n = n % 100; // 利用取余运算，计算剩下还需要支付的金额
    cout << n / 50 << endl; // 计算50元面额的张数并输出
    n = n % 50; // 更新剩下还需要支付的金额
    cout << n / 20 << endl; // 计算20元面额的张数并输出
    n = n % 20; // 更新剩下还需要支付的金额
    cout << n / 10 << endl; // 计算10元面额的张数并输出
    n = n % 10; // 更新剩下还需要支付的金额
    cout << n / 5 << endl; // 计算5元面额的张数并输出
    n = n % 5; // 更新剩下还需要支付的金额
    cout << n / 1 << endl; // 剩下的金额全为1元面额，直接计算并输出
    
    return 0;
}
```

## [P36. 练8.1 尼克与强盗](https://kuaima.nz/p/P36 "null")

**题目说明：** 输入一个两位数，交换十位与个位并输出。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 定义一个整型变量n，表示输入的两位数门牌号
    cin >> n; // 读取输入的门牌号数值
    int a = n / 10; // 使用除以10的方法获取十位上的数字
    int b = n % 10; // 使用对10取余的方法获取个位上的数字
    cout << b << a << endl; // 直接按顺序输出个位和十位上的数字，达到交换的效果
    
    return 0;
}
```

## [P41. 【例9.1】输出字符三角形](https://kuaima.nz/p/P41 "null")

**题目说明：** 给定一个字符，希望你用它构造一个底边长5个字符，高3个字符的等腰字符三角形。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    char c; // 定义一个字符型变量c，用来存储输入的字符
    cin >> c; // 读取用户输入的符号
    cout << "  " << c << endl; // 输出第一行，前面有两个空格，然后是1个字符
    cout << " " << c << c << c << endl; // 输出第二行，前面有一个空格，紧接着输出3个相同的字符
    cout << c << c << c << c << c << endl; // 输出第三行，没有空格，直接输出5个相同的字符
    
    return 0;
}
```

## [【例9.2】打印字符 - 题目详情 - Hydro](https://kuaima.nz/p/P42)

**题目说明：** 我们知道每个字符都有一个对应的 ASCII 码。现在输入一个 ASCII 码，要求你输出对应的字符。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 定义一个整型变量n，用来存储输入的ASCII码数值
    cin >> n; // 读取输入的数字（即ASCII码）
    char c = n; // 将整型的ASCII码赋值给字符型变量c，自动完成整数到字符的转换
    cout << c << endl; // 输出转换后得到的对应可见字符
    
    return 0;
}
```

## [P43. 【例9.3】小写字母转大写字母](https://kuaima.nz/p/P43 "null")

**题目说明：** 输入一个小写字母，则变为大写输出。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    char c; // 定义字符变量c，用来存储输入的小写字母
    cin >> c; // 读取输入的小写字母
    c = c - 'a' + 'A'; // 计算当前字母相对于'a'的偏移量，加上'A'的ASCII码即可转换为对应的大写字母
    cout << c << endl; // 输出转换后的大写字母
    
    return 0;
}
```
