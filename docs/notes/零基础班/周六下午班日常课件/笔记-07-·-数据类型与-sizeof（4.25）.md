# 笔记 07 · 数据类型与 sizeof（4.25）


该次课程，我们主要学习了一些基本数据类型所占的存储空间大小。该类型的题目在后续学习中主要以选择题的形式考察，不过我们需要知道如何使用C++代码测试出我们想要的基本数据类型。
#### 主要代码\函数
我们使用`sizeof()`函数可以非常方便的得到a变量类型所占的存储空间大小。不过注意这边的单位是`Byte`
代码示例如下：
```cpp
int a;
cout << sizeof(a);
```
变量a的类型是int，所以我们得到结果：4
#### 记忆内容
虽然这种方式我们不常用，不过，我们仍然需要记忆一些内容，首先是单位转化：
##### 基础单位：比特与字节
- **1 Byte (B, 字节) = 8 bits (b, 比特)**
    - _延伸说明：_ **bit (b)** 是计算机数据存储的最小单位（只能表示 0 或 1）。**Byte (B)** 是计算机数据处理的基本单位。
我们使用sizeof得到的单位也是`B`，这也计算机存储的基础单位。下面的单位都以该单位为基底，可以类比为米在长度单位中的地位。
##### 常规单位换算（按 1024 进制）
- **1 KB** (Kilobyte, 千字节) = **1024 B**
- **1 MB** (Megabyte, 兆字节) = **1024 KB**
- **1 GB** (Gigabyte, 吉字节) = **1024 MB**
- **1 TB** (Terabyte, 太字节) = **1024 GB** 
##### 64位系统常用数据类型内存占用表 

| 数据类型        | 含义     | 64位 Windows |
| :---------- | :----- | :---------: |
| `bool`      | 布尔型    |   1 Byte    |
| `char`      | 字符型    |   1 Byte    |
| `short`     | 短整型    |   2 Bytes   |
| `int`       | 整型     |   4 Bytes   |
| `long long` | 双长整型   |   8 Bytes   |
| `float`     | 单精度浮点型 |   4 Bytes   |
| `double`    | 双精度浮点型 |   8 Bytes   |
