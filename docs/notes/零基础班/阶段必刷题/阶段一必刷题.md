## 1. 输出语句 `cout`

使用 `cout` 将文字、数字或表达式的结果显示到屏幕上。`endl` 表示换行，每条输出语句末尾的分号不能省略。

**语法示例：**
```cpp
#include <iostream>
using namespace std;
int main() {
    cout << "Hello World" << endl;   // 输出一行并换行
    cout << 3 + 5 << endl;           // 输出表达式结果
    return 0;
}
```

**注意事项：**
- `cout << ""` 双引号里的内容会**原封不动**输出，不要往里面写变量名。
- `cout << a << b` 变量之间必须用 `<<` 分隔，不能写成 `cout << a b`，也不能用逗号。
- 每条语句末尾的**分号 `;` 不能漏**，这是初学者最高频的错误。
- `cout` 和 `<<` 之间、`<<` 和变量之间加空格不是必须的，但可以让代码更易读。

## 2. 变量与赋值

变量是用来存储数据的"盒子"，用 `int` 声明整型变量。`=` 表示赋值——把右边的值存到左边的变量里。通过第三个变量 `t` 可以交换两个变量的值。

**语法示例：**
```cpp
int a = 10;          // 声明整型变量并初始化
int b;
b = 20;              // 赋值
int t = a;
a = b;
b = t;               // 交换两个变量的值
```

**注意事项：**
- `=` 是**赋值**，不是数学中的"等于"。`a = a + 1` 的含义是把 a 的值加 1 再存回 a。
- 变量必须**先声明后使用**，未声明的变量名编译器不认识。
- 交换两个变量的值**必须借助第三个变量**，直接 `a = b; b = a;` 是错的（想一想为什么？）。
- 变量名不能以数字开头，不能有空格，不能用关键字（如 `int`、`return`）。

## 3. 输入语句 `cin`

使用 `cin` 从键盘读取用户输入的数据，`>>` 指向要存入的变量。可以连续读取多个值，用户输入时用空格或回车分隔。

**语法示例：**
```cpp
int a, b;
cin >> a >> b;       // 从键盘读入两个整数
cout << a + b;
```

**注意事项：**
- `cin >> a` 的箭头 `>>` 指向变量，方向是**从 cin 流向变量**，记住箭头指向你要存入的变量名。
- `cin >> a >> b` 读多个值时，输入用**空格或回车**隔开，不能用逗号。
- `cin` 读到空格或换行会自动分割，所以不要一次把带空格的话读进一个变量。
- 输入前最好用 `cout` 给个提示（如 `cout << "请输入年龄：";`），否则用户对着空屏幕不知道要输什么。

## 4. 算术运算 `+` `-` `*` `/` `%`

加 `+`、减 `-`、乘 `*` 与数学中一致。整数除法 `/` 会向下取整（舍去小数部分），`%` 是取余运算（求除法后的余数），只能用于整数。

**语法示例：**
```cpp
int sum = a + b;      // 加法
int diff = a - b;     // 减法
int prod = a * b;     // 乘法
int quot = a / b;     // 整数除法（舍去小数部分）
int rem = a % b;      // 取余（模运算，仅用于整数）
```

**注意事项：**
- 乘号必须写 `*`，不能写 `×`，也不能像数学那样省略（`2a` 在 C++ 中不合法，须写 `2 * a`）。
- 整数 `/` 整数，结果还是**整数**（小数部分直接丢弃，不是四舍五入）。例如 `5 / 2` 结果是 `2`，不是 2.5。
- `%` 只能用于**整型**，不能对浮点数取余。
- 运算有优先级：先乘除后加减，有括号先算括号。不确定时**多加括号**是最安全的做法。

## 5. 浮点数与小数保留

`double` 用于存储带小数点的数。用 `fixed` 与 `setprecision(n)` 配合，可以控制输出时保留 n 位小数。记得添加头文件 `#include <iomanip>`。

**语法示例：**
```cpp
#include <iomanip>
double r, area;
cin >> r;
area = 3.14159 * r * r;
cout << fixed << setprecision(3) << area;   // 保留 3 位小数
```

**注意事项：**
- 需要小数的变量用 `double`，不要用 `int`——`int` 存不下小数部分。
- `setprecision` 必须配合 `fixed` 使用才能**固定小数位数**；不加 `fixed` 时 `setprecision(n)` 控制的是**有效数字位数**。
- `fixed` 和 `setprecision` 对之后所有 `cout` 输出都生效，一般写一次就行。
- 不要忘记 `#include <iomanip>`，否则编译报错。

## 6. 字符类型 `char` 与 ASCII

`char` 存储单个字符，每个字符在计算机内部对应一个 ASCII 码（整数）。用 `int(c)` 可读取该码值，用 `char(n)` 可将整数转回字符。大写字母 `'A'`（65）与小写字母 `'a'`（97）相差 32。

**语法示例：**
```cpp
char c;
cin >> c;
cout << char(c + 1) << endl;   // ASCII 码 +1 后的字符
cout << int(c) << endl;        // 输出字符对应的 ASCII 码
```

**注意事项：**
- 字符用**单引号** `'A'`，字符串用**双引号** `"Hello"`。单引号里只能放一个字符。
- `int(c)` 得到字符对应的 ASCII 码；`char(65)` 得到整数 65 对应的字符 `'A'`。
- 大小写转换：大写 + 32 = 小写，小写 - 32 = 大写。`'a' - 'A'` 的结果就是 32。
- `char` 变量用 `cin` 读入时会跳过空白字符（空格、换行）。

## 7. 数据类型与 `sizeof`

C++ 中不同数据类型占用不同的内存空间。`sizeof(类型)` 可查看某个类型占多少字节。常用类型：`int`（整型）、`double`（双精度浮点）、`char`（字符）、`bool`（布尔，仅 `true`/`false`）。

**语法示例：**
```cpp
cout << sizeof(int) << endl;     // 4（不同机器可能不同）
cout << sizeof(double) << endl;  // 8
cout << sizeof(char) << endl;    // 1
cout << sizeof(bool) << endl;    // 1
bool flag = true;                // 布尔型只有 true / false
```

**注意事项：**
- `sizeof` 不是函数，是 C++ 关键字，括号里可以直接写类型名或变量名。
- `bool` 类型只有两个值：`true`（1）和 `false`（0）。输出 `bool` 时默认显示 1 或 0。
- `int` 的大小不一定是 4 字节（不同平台不同），但在 OJ 评测机上通常是 4 字节。
- 选择数据类型时要考虑范围：`int` 最大约 21 亿，超出要用 `long long`。

## 8. 格式化输入输出

**注意事项：**
- `setprecision` 控制的是小数位数，不是总位数。

## 9. `if` 单分支判断

`if` 语句让程序根据条件决定是否执行一段代码。条件写在圆括号中，条件成立（为 `true`）时执行花括号中的内容；不成立则直接跳过。

**语法示例：**
```cpp
if (n > 0) {
    cout << "positive" << endl;   // 条件成立时执行
}
if (n % 2 == 0) {
    cout << "even" << endl;
}
```

**注意事项：**
- 条件必须写在**圆括号 `()`** 里。
- 即使 `if` 后面只有一条语句，也建议写**花括号 `{}`**——偷懒不写是很多 bug 的来源。
- 判断相等用 `==`，**不是 `=`**。`if (a = 5)` 是把 5 赋给 a 再判断，结果永远是 `true`——这是大家高频的错误之一。
- `if` 的条件可以是一个 `bool` 变量：`if (flag)` 等价于 `if (flag == true)`。

## 10. `if-else` 双分支

`if-else` 处理"二选一"——条件成立执行 `if` 分支，不成立执行 `else` 分支。**两条路径必走其一**。

**语法示例：**
```cpp
if (n % 2 == 0) {
    cout << "偶数" << endl;     // 条件成立
} else {
    cout << "奇数" << endl;     // 条件不成立
}
```

**注意事项：**
- `else` 后面**不写条件**，它自动取 `if` 条件的反情况。`else (n % 2 != 0)` 是错的。
- `else` 不能单独出现，必须跟在 `if` 或 `else if` 后面。
- `if-else` 是一条语句，中间不能再插别的语句。
- 如果你的 `else` 分支什么也不做，那说明可能不需要 `else`。

## 11. `else if` 多分支

当有多个互斥的条件需要逐一判断时，在 `if` 后面用 `else if` 串联。程序从上往下依次检查，命中第一个成立的分支后跳过剩余判断。末尾的 `else` 捕捉所有未被命中情况。

**语法示例：**
```cpp
if (score >= 90) {
    cout << "A";
} 
else if (score >= 80) {
    cout << "B";
} 
else if (score >= 60) {
    cout << "C";
} 
else {
    cout << "D";
}
```

**注意事项：**
- `else if` 后面**必须跟条件**（圆括号里写判断），而最后的 `else` 不跟条件。
- 条件判断是**从上往下依次检查**的，所以顺序很重要。上面例子中 `score >= 80` 能写在 `score >= 90` 后面，就是因为 90 分以上的情况已经在第一个分支被截住了。
- `else` 只能放在最后，且**最多一个**；`else if` 可以有任意多个。
- 如果分支间不是互斥的，不要用 `else if`，应该写成多个独立的 `if`。

## 12. 比较运算符与嵌套 `if`

比较运算符（`>`、`<`、`>=`、`<=`、`==`、`!=`）用于比较两个值，结果是 `true` 或 `false`。注意判断相等用 `==`，不是 `=`。`if` 内部可以再写 `if`，形成嵌套判断。

**语法示例：**
```cpp
if (a > b) {                      // 比较运算符：> < >= <= == !=
    if (a > c) {                  // 嵌套 if
        cout << a << " is max";
    }
}
```

**注意事项：**
- `==` 是判断相等，`=` 是赋值。**千万不要混用**，`if (a = 10)` 编译器通常不报错但逻辑全错。
- `!=` 是"不等于"，`>=` 是"大于等于"，`<=` 是"小于等于"。写法顺序固定，不能写成 `=<` 或 `=>`。
- 嵌套 `if` 时注意花括号配对——哪个 `{` 属于哪个 `}`，缩进要对齐，否则逻辑一团乱。
- 三个数比大小可以用嵌套 `if`，也可以先假设第一个最大，再和后面的逐个比较。

## 13. 逻辑运算符 `&&` `||` `!`

`&&`（与）要求两侧条件同时成立；`||`（或）要求至少一侧成立；`!`（非）将真变假、假变真。逻辑运算符用来组合多个简单条件，形成复合判断。

**语法示例：**
```cpp
if (x >= 1 && x <= 10)  { ... }      // 与：两个条件都成立
if (a > 100 || b > 100) { ... }      // 或：至少一个成立
if (!flag) { ... }                    // 非：取反
```

**注意事项：**
- `&&` 是"且"（and），**两侧都必须满足**。写 `1 <= x <= 10` 是错的，必须写 `x >= 1 && x <= 10`。
- `||` 是"或"（or），**只要一边满足**整个条件就成立。
- `!` 写在条件前面表示"非"。`!flag` 表示 flag 为假时条件成立。
- `&&` 优先级高于 `||`，不确定时**多加括号**：`(a && b) || c` 和 `a && (b || c)` 结果完全不同。

## 14. 复合条件判断

将比较运算符和逻辑运算符结合使用，可以表达更复杂的判断逻辑。括号 `()` 用于明确优先级——就像数学中先算括号里的内容一样。典型例子：闰年判断。

**语法示例：**
```cpp
if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0) {
    cout << "leap year";   // 闰年条件：被 4 整除且不被 100 整除，或被 400 整除
}
```

**注意事项：**
- 复杂条件**多加括号**，先保证结果正确，再考虑简化。因省略括号搞错优先级是最亏的 bug。
- 像闰年这种条件，先想清楚逻辑再写代码：能被 400 整除 → 闰年；能被 4 整除但不能被 100 整除 → 闰年；其余 → 平年。
- 条件写完后用几个已知答案的数据（如 2000、1900、2024）**人工验算一下**，确保逻辑无误。

## 15. `switch` 语句

当一个变量有多个明确的取值、每种取值对应不同操作时，`switch` 比多层 `else if` 更清晰。每个 `case` 对应一种值，`break` 用于跳出（否则会继续执行下一个 `case`）。`default` 处理未列出的情况。

**语法示例：**
```cpp
switch (day) {
    case 1: cout << "Monday"; break;
    case 2: cout << "Tuesday"; break;
    default: cout << "Invalid"; break;   // 都不匹配时执行
}
```

**注意事项：**
- `switch` 括号里必须是**整数或字符**（`int`、`char`），不能是 `double` 或 `string`。
- `case` 后面是**常量**（如 `1`、`'A'`），不能写变量或表达式。
- 每个 `case` 末尾的 **`break` 绝对不能忘**——否则会"穿透"到下一个 `case` 继续执行，造成逻辑错误。
- `default` 写在最后，处理所有没有列出的情况。即使看起来"不可能走到"，也建议保留。
