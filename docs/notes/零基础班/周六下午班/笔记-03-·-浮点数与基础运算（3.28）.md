# 笔记 03 · 浮点数与基础运算

## [P24. 练6.1 植树造林](https://kuaima.nz/p/P24)

**题目说明：** 某小学向全校师生发出“植树造林，还我绿色”的倡议，鼓励大家多植树。给出每个班的人数与平均每个人种的棵数，输出这个班总共种了几棵树。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n, m; // 定义两个整数变量n和m，分别表示人数和平均每人种的棵数
    cin >> n >> m; // 从输入中读取人数和每人种的棵数，存入变量
    int total = n * m; // 计算总棵数，即人数乘以平均每人种的棵数
    cout << "总的棵数：" << total << endl; // 按照题目要求的格式输出文本和计算出的总棵数

    return 0;
}
```

## [P27. 【例7.1】保留3位小数](https://kuaima.nz/p/P27)

**题目说明：** 读入一个浮点数，保留3位小数输出这个浮点数。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double x; // 定义一个双精度浮点型变量x，用来存储带小数的数字
    cin >> x; // 获取用户输入的浮点数并赋值给x
    cout << fixed << setprecision(3) << x; // fixed和setprecision(3)组合使用，表示将x保留3位小数输出

    return 0;
}
```

## [P28. 【例7.2】与圆相关的计算](https://kuaima.nz/p/P28)

**题目说明：** 给出圆的半径，求圆的直径、周长和面积。约定 $\pi = 3.14159$。保留4位小数。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double r; // 定义双精度浮点变量r，代表圆的半径
    cin >> r; // 读取题目给出的半径数值
    double pi = 3.14159; // 按照题目要求，定义并初始化圆周率pi的值
    double d = 2 * r; // 计算圆的直径，公式为：半径乘以2
    double c = 2 * pi * r; // 计算圆的周长，公式为：2 * pi * r
    double s = pi * r * r; // 计算圆的面积，公式为：pi * r的平方
    cout << fixed << setprecision(4); // 设置后面的所有浮点数输出都固定保留4位小数
    cout << d << " " << c << " " << s << endl; // 依次输出直径、周长和面积，中间用一个空格隔开

    return 0;
}
```

## [P29. 练7.1 埃及金字塔](https://kuaima.nz/p/P29)

**题目说明：** 给出三角形的底和高，输出其中一个侧面的面积。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double a, h; // 定义两个双精度浮点型变量a和h，分别代表底和高
    cin >> a >> h; // 从输入中读取底和高的数值
    double area = a * h / 2.0; // 利用三角形面积公式(底乘高除以2)计算面积，除以2.0保证精度
    cout << "s=" << area << endl; // 先输出字符"s="，紧接着输出计算得到的面积值

    return 0;
}
```

## [P31. 练7.3 买图书](https://kuaima.nz/p/P31)

**题目说明：** 已知小明有 $n$ 元，他买了一本书，这本书原价为 $m$ 元，现在打8折出售。求小明还剩多少钱(保留2位小数)。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double n, m; // 定义两个双精度浮点型变量，n表示小明的总钱数，m表示书的原价
    cin >> n >> m; // 分别读取总钱数和书的原价
    double cost = m * 0.8; // 计算打8折后书的实际售价，即原价乘以0.8
    double remain = n - cost; // 计算小明买书后剩余的钱，即总钱数减去实际售价
    cout << fixed << setprecision(2) << remain; // 按照题目要求，将剩余的钱数保留2位小数输出

    return 0;
}
```
