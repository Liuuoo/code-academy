# 笔记 08 · 条件判断（5.4）


## [P84. 练15.2 智商问题](https://kuaima.nz/p/P84 "null")

**题目说明：** 智商（IQ）反映人的聪明程度，它是法国心理学家比奈提出的。他将一般人的平均智商定为100。分数越高，表示越聪明，智商就越高，140分以上者称为天才。试编一程序，输出一个200以内的整数作为IQ值，判断是不是天才。如果满足条件则输出“天才”，否则不输出。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int iq; // 定义一个整型变量iq，用于存储输入的智商数值
    cin >> iq; // 从输入中读取智商数值，并赋值给变量iq
    if (iq >= 140) // 判断读取到的智商数值是否大于或等于140
    {
        cout << "天才" << endl; // 如果智商达到140及以上，输出“天才”文本并换行
    }
    
    return 0;
}
```

## [P82. 【例15.3】 最大数输出](https://kuaima.nz/p/P82 "null")

**题目说明：** 输入三个整数，数与数之间以一个空格分开。 输出一个整数，即最大的整数。（三个数两两互不相同）

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a, b, c; // 定义三个整型变量a、b、c，用来存储输入的三个整数
    cin >> a >> b >> c; // 依次从输入中读取这三个整数
    int max_num = max(a, max(b, c)); // 嵌套使用内置的max函数，先求出b和c的最大值，再将其与a比较，得出最大值
    cout << max_num << endl; // 输出最终找到的最大整数并换行
    
    return 0;
}
```

## [练15.3 最大的数 - 题目详情 - Hydro](https://kuaima.nz/p/P85)

**题目说明：** 输入三个数（可能包含小数），输出其中最大的数。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double a, b, c; 
    cin >> a >> b >> c;
    double max_num = max(a, max(b, c)); // 同样嵌套使用内置的max函数求出三个数中的最大值
    cout << max_num << endl; // 输出最终找到的最大数值并换行
    
    return 0;
}
```

**补充说明：** 以上关于“最大数输出”的题目演示了如何使用C++中便捷的 `max` 函数来快速求解。但是，为了巩固基础知识，**课后作业要求大家务必使用课上先讲解的 if 语句（条件分支语句）来重新编写并完成这两道题**。请通过纯粹的逻辑判断来找出最大值。
