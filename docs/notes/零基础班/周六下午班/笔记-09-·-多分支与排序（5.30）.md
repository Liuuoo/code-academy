# 笔记 09 · 多分支与排序（5.30）


## [P109. 练19.2 三个数](https://kuaima.nz/p/P109 "null")

**题目说明：** 输入三个整数，按从大到小的顺序输出。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a, b, c;
    cin >> a >> b >> c;
    if (a < b) // 判断第一个数a是否小于第二个数b
    {
        swap(a, b); // 如果a小于b，则交换它们的值，保证a是两者中较大的那个
    }
    if (a < c) // 判断当前a是否小于第三个数c
    {
        swap(a, c); // 如果a小于c，再次交换，此时a一定为三个数中的最大值
    }
    
    // a一定为三个数中的最大值，但b位置上的数不一定是次大。
    
    if (b < c) // 判断第二个数b是否小于第三个数c
    {
        swap(b, c); // 如果b小于c，交换它们，使得b为次大值，c为最小值
    }
    cout << a << " " << b << " " << c << endl; // 按照从大到小的顺序，用空格隔开输出排好序的三个数
    
    return 0; // 程序正常结束
}
```
该题解思想为选择排序思想，将最大数放到第一，次大放到第二位...
## [P108. 练19.1 整数大小比较](https://kuaima.nz/p/P108 "null")

**题目说明：** 输入两个整数，比较它们的大小。若x>y，输出>；若x=y，输出=；若x<y，输出<。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x, y; 
    cin >> x >> y;
    if (x > y) // 判断变量x的值是否大于变量y的值
    {
        cout << ">" << endl; // 如果大于，则按照题目要求输出大于号
    }
    else if (x == y) // 否则，继续判断变量x的值是否等于变量y的值
    {
        cout << "=" << endl; // 如果等于，则输出等于号
    }
    else // 如果上面两种情况都不满足，说明x一定小于y
    {
        cout << "<" << endl; // 那么直接输出小于号即可
    }
    
    return 0; // 程序正常结束
}
```

## [P102. 练18.2 苹果和虫子](https://kuaima.nz/p/P102 "null")

**题目说明：** 买了一箱n个苹果，虫子每x小时能吃掉一个苹果。经过y小时后，还剩下多少个完整的苹果？

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n, x, y;
    cin >> n >> x >> y;
    if (y % x == 0) // 判断经过的时间除以吃苹果的时间是否没有余数，说明没有正在吃（多吃一口）的情况
    {
        if (n - (y / x) < 0) // 判断如果虫子吃掉的苹果数量超过了拥有的苹果总数（时间过多，没得吃了）
        {
            cout << 0 << endl; // 苹果数量不能为负数，直接输出0
        }
        else // 否则说明苹果还没有吃完，或者正好吃完
        {
            cout << n - (y / x) << endl; // 输出总苹果数减去被完全吃掉的苹果数量
        }
    }
    else // 如果有余数，说明虫子多吃了一口（即有一个苹果被咬了，不完整了）
    {
        if (n - (y / x + 1) < 0) // 判断被完全吃掉的加上被破坏的这1个苹果是否超过了总数
        {
            cout << 0 << endl; // 如果超过了，剩下完整的苹果也只能是0个
        }
        else // 否则说明还有剩余的完整苹果
        {
            cout << n - (y / x + 1) << endl; // 输出总苹果数减去（完全吃掉的个数加上被咬坏的1个）
        }
    }
    
    return 0;
}
```

## [P94. 【例17.3】 内功逼毒](https://kuaima.nz/p/P94 "null")

**题目说明：** 黄蓉在t时间内不解毒会身亡。郭靖每个单位时间可以逼出m点毒，黄蓉一共中了n点毒。判断黄蓉能否得救，得救输出yes，否则输出no。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int t, n, m;
    cin >> t >> n >> m;
    int total = t * m; // 计算在允许的最大时间内，郭靖一共能够逼出的毒量总和
    if (total >= n) // 判断总逼毒量是否大于或等于黄蓉中的总毒量
    {
        cout << "yes" << endl; // 如果逼毒量足够解毒，则黄蓉能得救，输出yes
    }
    else // 否则（即逼毒总量小于总毒量）
    {
        cout << "no" << endl; // 说明在规定时间内无法逼完毒，黄蓉不能得救，输出no
    }
    
    return 0; // 程序正常结束
}
```
