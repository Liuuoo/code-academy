# 笔记 06 · 字符与类型转换复习（4.18）



**课程说明：** 这节课是复习课，所以没有新知识点，主要通过实战练习巩固我们之前学过的变量、基本运算和类型转换等基础知识。
## [P42. 【例9.2】打印字符](https://kuaima.nz/p/P42 "null")

**题目说明：** 我们知道每个字符都有一个对应的 ASCII 码。现在输入一个 ASCII 码，要求你输出对应的字符。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a; // 定义一个整数变量a，用来接收输入的ASCII码
    cin >> a; // 读取用户输入的整数
    char c = a; // 将整数赋值给字符型变量c，实现ASCII码到字符的转换
    cout << c << endl; // 输出转换后对应的可见字符
    
    return 0;
}
```

## [P43. 【例9.3】小写字母转大写字母](https://kuaima.nz/p/P43 "null")
**题目说明：** 输入一个小写字母，则变为大写输出。
**解题思路（偏移量法）：** 在ASCII码表中，小写字母 `a~z` 和大写字母 `A~Z` 分别是连续排列的。因此，我们可以先计算出输入的小写字母距离首字母 `'a'` 有多远（即偏移量 `d = c - 'a'`），然后将大写字母 `'A'` 加上同样的距离（即 `'A' + d`），就能精准地得到对应的大写字母了。这种方法不需要死记硬背数字，逻辑更清晰。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    char c; // 定义一个字符变量c，用于存储输入的小写字母
    cin >> c; // 读取输入的小写字母
    int d = c - 'a'; // 计算输入字母相对于小写字母'a'的偏移量d
    char upper = 'A' + d; // 将大写字母'A'加上这个偏移量d，即可得到对应的大写字母
    cout << upper << endl; // 输出计算后得到的大写字母
    
    return 0;
}
```

## [P47. 【例10.1】机票打折](https://kuaima.nz/p/P47 "null")

**题目说明：** 输入机票原价，再输入机票打折率。编程计算打折后机票的实际价格。根据要求，本题直接使用 `a * b / 10` 的方式进行打折计算，不使用题目原定的复杂四舍五入规则。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double a, b; // 定义两个双精度浮点型变量a和b，分别表示机票原价和打折率
    cin >> a >> b; // 获取输入的机票原价与打折率
    double price = a * b / 10.0; // 按照要求直接使用 a*b/10 计算打折后的价格
    int ans = price; // 将浮点数价格赋值给整数变量，去除小数部分
    cout << ans << endl; // 输出最终的整数价格
    
    return 0;
}
```

## [P49. 练10.1 细胞分裂](https://kuaima.nz/p/P49 "null")

**题目说明：** 1个细胞，第1次分裂成2个，第2次分裂成4个……试编一程序，算一算第5次分裂成几个？

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int cell = 1; // 定义一个整数变量cell，表示最开始的细胞数量为1
    cell = cell * 2; // 第1次分裂，细胞数量乘以2
    cell = cell * 2; // 第2次分裂，细胞数量再次乘以2
    cell = cell * 2; // 第3次分裂，继续乘以2
    cell = cell * 2; // 第4次分裂，继续乘以2
    cell = cell * 2; // 第5次分裂，最后一次乘以2
    cout << cell << endl; // 输出经过5次分裂后的细胞总数
    
    return 0;
}
```

## [P50. 练10.2 浮点数向零舍入](https://kuaima.nz/p/P50 "null")

**题目说明：** 输入一个浮点数，将其向零舍入到整数。说明：向零舍入的含义是，正数向下舍入，负数向上舍入。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double x; // 定义一个双精度浮点型变量x，用于接收输入的小数
    cin >> x; // 读取输入的浮点数
    int ans = x; // 将浮点数赋值给整型变量，C++中此操作会自动截断小数部分，完美实现向零舍入
    cout << ans << endl; // 输出舍入后的整数结果
    
    return 0;
}
```
