# 笔记 10 · 逻辑运算符（6.6）


本节课主要复习前面题目，然后学习了与或非这三个逻辑运算符。

| 运算符 | C++ 符号 | 英文名称 | 逻辑关系    | 真值判定规则                          |
| --- | ------ | ---- | ------- | ------------------------------- |
| 与   | &&     | AND  | 并且      | 所有条件都为真时，结果才为真；只要有一个为假，结果就为假。   |
| 或   | &#124; | OR   | 或者      | 只要有一个条件为真，结果就为真；所有条件都为假时，结果才为假。 |
| 非   | !      | NOT  | 取反 / 不是 | 将真变成假（0），将假（0）变成真（1）。           |
他们的优先级关系是：非与或。可以使用非与或来记（飞鱼或）
## [P113. 【例20.1】 范围判断](https://kuaima.nz/p/P113 "null")

**题目说明：** 读入一个整数，若这个数大于1并且小于100，则输出yes。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    cin >> n;
    if (n > 1 && n < 100) // 使用逻辑与运算符&&判断n是否同时满足大于1且小于100
    {
        cout << "yes" << endl; // 若条件成立，向标准输出打印"yes"并换行
    }

    return 0; // 主函数结束，向操作系统返回0表示程序正常运行完毕
}
```

## [P114. 【例20.2】 收集瓶盖赢大奖](https://kuaima.nz/p/P114 "null")

**题目说明：** 某饮料公司最近推出了一个"收集瓶盖赢大奖"的活动：如果你拥有10个印有"幸运"、或20个印有"鼓励"的瓶盖，就可以兑换一个神秘大奖。现分别给出你拥有的印有"幸运"和"鼓励"的瓶盖数，判断是否可以去兑换大奖。若可以兑换大奖，输出1，否则输出0。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int luck, encourage; 
    cin >> luck >> encourage; 
    if (luck >= 10 || encourage >= 20) // 使用逻辑或运算符||判断是否满足任意一个兑奖条件
    {
        cout << 1 << endl; // 若满足条件，输出数字1并换行
    }
    else
    {
        cout << 0 << endl; // 若两个条件都不满足，输出数字0并换行
    }

    return 0; // 主函数结束，向操作系统返回0表示程序正常运行完毕
}
```

## [P115. 【例20.3】 0与1](https://kuaima.nz/p/P115 "null")

**题目说明：** 第一堂计算机概论课上，老师说计算机内部是由很多的小开关来组成：1 代表开、0 代表关。输入只有一行，含有一个为 0 或 1 的整数。输入为 0 则输出 1；输入为 1 则输出 0。

**代码实现：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x; 
    cin >> x; 
    cout << !x << endl; // 使用逻辑非运算符!对x进行取反操作，0取反为1，1取反为0，并将结果输出换行

    return 0; // 主函数结束，向操作系统返回0表示程序正常运行完毕
}
```
