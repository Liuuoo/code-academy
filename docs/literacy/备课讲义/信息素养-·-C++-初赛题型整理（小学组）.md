# 信息素养 · C++ 初赛题型整理（小学组）


> 依据目录中的初赛模拟题与 2025 初赛真题整理。
> 目标：按“知识点 + 典型问题 + 例题”形成备课提纲，便于课堂讲解与刷题训练。

---

## 一、初赛整体特点

初赛题目以**选择题、判断题、读程序写结果、补全代码**为主，重点不是大规模算法实现，而是：

1. **C++ 语法基础是否熟练**
2. **是否能读懂短程序**
3. **是否理解数据类型、运算规则、分支循环**
4. **是否会处理数组、字符串、简单结构体**
5. **是否具备基础计算机常识**

建议备课时按下面几类展开。

---

# 二、题型分类整理

## 1. 计算机基础常识题

### 知识点
- 输入设备、输出设备、存储设备
- Bit / Byte 等基础单位
- 操作系统常识
- 快捷键、复制粘贴、剪切等基本操作
- 程序语言与 CPU 执行方式的基本理解

### 典型问题
- 下列设备中哪个是输入设备/输出设备？
- 存储容量的基本单位是什么？
- Ctrl + X / Ctrl + V / Ctrl + C 分别表示什么？
- 高级语言编写的程序能否被 CPU 直接执行？

### 例题
1. 下列设备中，属于输出设备的是：显示器 / 键盘 / 鼠标 / 麦克风。  
2. Windows 中，`Ctrl + X` 的作用是什么？  
3. 存储容量单位中，最基本的是 `bit` 还是 `byte`？

### 备课提醒
- 这类题分值稳定，适合课前热身。
- 学生容易混淆输入输出设备、bit 和 byte。

---

## 2. 标识符、关键字与注释

### 知识点
- 标识符命名规则
  - 只能由字母、数字、下划线组成
  - 不能以数字开头
  - 不能和关键字重名
- 常见关键字：`int`、`if`、`for`、`while`、`do`、`else`
- 注释写法：`//`、`/* ... */`

### 典型问题
- 哪个是合法标识符？
- 哪个是 C++ 关键字？
- 哪个是正确注释？

### 例题
1. 下列可作为 C++ 标识符的是：`x321`、`0x321`、`Man&Woman`、`break`。  
2. 下列哪个是 C++ 中合法的单行注释？  
   - `// 注释内容`
   - `# 注释内容`
   - `(* 注释内容 *)`
   - `<!-- 注释内容 -->`
3. 下列哪个不是关键字：`for`、`do`、`int`、`cout`。

### 备课提醒
- `cout`、`endl` 常被误认为关键字，实际不是。
- `break` 是关键字，不能作为变量名。

---

## 3. 数据类型与常量

### 知识点
- 基本类型：`int`、`double`、`float`、`char`、`bool`、`void`
- 字符常量与数字常量区别
  - `'3'` 是 `char`
  - `3` 是 `int`
- `bool` 的逻辑意义
- `void` 表示无返回值

### 典型问题
- `7.0` 属于什么类型？
- `'3'` 是什么类型？
- `bool`/`void` 的含义是什么？
- 哪些函数可以没有返回值？

### 例题
1. 常量 `7.0` 的类型是：`double` / `float` / `void` / `int`。  
2. `'3'` 的数据类型是：`char` / `int` / `float` / `double`。  
3. C++ 中是否可以定义无返回值函数？

### 备课提醒
- 初赛很喜欢考 `'9' * 3`、`'0'` 的 ASCII 相关问题。
- `char` 参与运算时会先转成整数。

---

## 4. 进制与字符编码

### 知识点
- 十进制、八进制、十六进制表示
- 八进制以 `0` 开头
- 十六进制以 `0x` 开头
- 字符的 ASCII 编码

### 典型问题
- `02023` 表示几进制？
- `0x212` 转成十进制是多少？
- `'9' * 3` 的结果是多少？

### 例题
1. 已知 `int a = 02023, b = 0x212;`，求 `a + b`。  
2. `cout << '9' * 3;` 输出什么？  
3. `'0'` 的 ASCII 码常见基准值是多少？

### 备课提醒
- 这是高频“陷阱题”。
- 可专门安排“进制与字符编码”小专题。

---

## 5. 运算符与表达式

### 知识点
- 算术运算：`+ - * / %`
- 整数除法与浮点除法
- 取模 `%` 只能用于整数
- 复合赋值：`+= -= *= /= %=`
- 自增自减：`++ --`
- 运算优先级与括号

### 典型问题
- `14 / 3 * 3` 结果是多少？
- `15 / 2.0` 和 `15 / 2` 有什么区别？
- `a *= 3`、`a %= 4` 后结果是多少？
- 哪个表达式能得到 `3.5`？

### 例题
1. `int a = 14 / 3 * 3;` 输出什么？  
2. 若 `a = 6`，执行 `a %= 4;` 后 `a` 为多少？  
3. 已知 `a = 7, b = 2`，哪个表达式的结果是 `3.5`？  
4. `13.8 % 2` 是否合法？为什么？

### 备课提醒
- 这里是初赛最高频考点之一。
- 要强调：
  - `int / int` 结果仍为整数
  - 只要有一个浮点数参与，结果一般为浮点数
  - `%` 不能用于小数

---

## 6. 逻辑表达式与条件判断

### 知识点
- 比较运算：`== != > < >= <=`
- 逻辑运算：`&& || !`
- `if`、`if-else`
- 判断“同时成立”“至少一个成立”
- 布尔值在表达式中的应用

### 典型问题
- 如何判断 `a` 和 `b` 都等于 0？
- 哪个 `if-else` 结构写法正确？
- 哪个表达式表示“a 和 b 都不为 0”？

### 例题
1. 判断 `a == 0` 且 `b == 0` 的正确写法是：  
   - `(a == 0) && (b == 0)`  
   - `(a == b == 0)`  
   - `(!a) && (!b)`  
2. 下列哪一个是完整正确的分支结构？  
3. 选择能判断“a 和 b 都非 0”的表达式。

### 备课提醒
- `a == b == 0` 是经典误区。
- 可以安排“真假值”和“逻辑运算优先级”专项训练。

---

## 7. 顺序、分支、循环三大程序结构

### 知识点
- 顺序结构
- 分支结构：`if / if-else`
- 循环结构：`for / while / do...while`
- 三大结构的概念区分

### 典型问题
- 程序控制的基本结构有哪些？
- 哪段代码属于分支结构？
- `do...while` 的特点是什么？

### 例题
1. C++ 程序控制的三种基本结构是什么？  
2. 下列代码中，哪一段体现了分支结构？  
3. 判断：`do...while` 循环至少会执行一次。

### 备课提醒
- 初赛常用概念题考查，不难，但容易失分。
- 建议和流程图、生活场景结合讲。

---

## 8. 循环与计数型程序

### 知识点
- `for` 循环的初始化、条件、步长
- 累加、计数、找最大值/最小值
- 循环边界
- 多次更新变量

### 典型问题
- 哪个循环能遍历数组全部元素？
- 代码执行若干次后输出什么？
- 如何在循环中统计满足条件的个数？

### 例题
1. `for (int i = 0; i < 4; i++) a[i] *= 2;` 的作用是什么？  
2. 某段循环执行后输出 `1248`，横线处该填什么？  
3. 某段程序输出 `147`，循环条件应如何补全？

### 备课提醒
- 小学组很喜欢考“补全循环代码”。
- 重点训练边界：从 `0` 开始还是从 `1` 开始，条件是 `< n` 还是 `<= n-1`。

---

## 9. 读程序写结果

### 知识点
- 按执行顺序模拟程序
- 跟踪变量变化
- 结合分支和循环判断最终输出

### 典型问题
- 给定输入，程序最后输出什么？
- 连续两个 `if` 与 `if...else` 的区别是什么？
- 每一步变量值如何变化？

### 例题
1. 输入 `6 7`：
   ```cpp
   int a, b;
   cin >> a >> b;
   if (a % 2 == 0) a /= 2;
   if (a % 2 != 0) a = a * 3 + 1;
   cout << a + b;
   ```
   输出是什么？
2. `int a = 14 / 3 * 3; cout << a;` 输出什么？
3. 多步复合赋值程序的最终结果是什么？

### 备课提醒
- 这是最能拉开差距的题型。
- 课堂上建议教学生列表跟踪：步骤、变量值、是否进分支。

---

## 10. 数组基础题

### 知识点
- 一维数组定义与访问
- 下标从 `0` 开始
- 数组元素遍历
- 越界问题
- 二维数组的初始化

### 典型问题
- 哪段代码能正确遍历数组？
- 数组下标写成 `1 ~ n` 会有什么问题？
- 二维数组默认补零后某个位置的值是多少？

### 例题
1. `int a[4] = {2,4,6,8};` 哪段代码能把每个元素变成原来的 2 倍？  
2. `int a[3][4] = {​{1,2},{3},{4,5,6,7}​};` 求 `a[1][2]` 和 `a[2][1]`。  
3. 下列循环中，哪个会造成数组越界？

### 备课提醒
- 下标越界几乎每年都会以选择题方式出现。
- `for (int i = 1; i <= 4; i++)` 这种错法要反复强调。

---

## 11. 字符串基础题

### 知识点
- `string` 的输入与长度
- 逐字符遍历
- 相邻字符比较
- 连续相同字符统计

### 典型问题
- 如何求字符串中最长连续相同字符段？
- `s[i]` 应该和谁比较？
- 从哪个下标开始遍历更合理？

### 例题
1. 给定 DNA 字符串，仅包含 `A、C、G、T`，求最长连续相同字符子串长度。补全：
   ```cpp
   string s;
   cin >> s;
   int len = __, ans = 1;
   for (int i = __; i < s.size(); i++) {
       if (s[i] == s[__]) len++;
       else len = 1;
       if (__) ans = len;
   }
   cout << ans;
   ```
2. 解释为什么这类题通常从 `i = 1` 开始遍历。

### 备课提醒
- 这是典型“代码补空 + 思维题”。
- 重点是“当前字符与前一个字符比较”。

---

## 12. 排序、比较与简单算法思想

### 知识点
- 比较类排序与非比较类排序
- 选择排序、冒泡排序、插入排序、桶排序的基础认知
- 非递减序列的概念
- 简单贪心/调整思路

### 典型问题
- 哪种排序不基于比较？
- 如何最少操作让数组变成非递减？
- 每次可以增加 1，最少操作数怎么求？

### 例题
1. 下列排序算法中，不使用“比较”思想的是：选择排序 / 冒泡排序 / 插入排序 / 桶排序。  
2. 给定数组，每次可将某个元素加 1，最少多少次操作能使其变成非递减序列？补全代码。  
3. 什么叫“非递减”？

### 备课提醒
- 初赛不会考复杂实现，但会考概念辨析和简单补代码。
- 讲题时要把“前一个比后一个大，就把后一个补到前一个”讲清楚。

---

## 13. 结构体基础题

### 知识点
- `struct` 的定义
- 成员访问运算符 `.`
- 结构体数组
- 结构体成员类型可以不同

### 典型问题
- 结构体能否包含不同类型成员？
- 结构体数组如何访问？
- 表达式 `s[0].d + s[1].i` 结果是什么？

### 例题
1. 已知：
   ```cpp
   struct S { double d; int i; };
   S s[2] = {{1.5, 1}, {2.5, 2}};
   ```
   求 `s[0].d + s[1].i`。  
2. 关于结构体，下列说法哪项正确？

### 备课提醒
- 常见考法是“概念判断 + 一小段读程序”。
- 重点说明：结构体不是只能放同类型数据。

---

## 14. 递归入门题

### 知识点
- 递归函数的定义
- 递归出口
- 递归展开与回溯求值

### 典型问题
- 函数什么时候停止调用？
- 给定 `func(10)`，最后结果是多少？
- 如何一层层代回去求值？

### 例题
1. 已知：
   ```cpp
   int func(int x)
   {
       if (x <= 4) return 2 * x - 1;
       else if (x > 7) return func(x - 4) + x;
       else return func(x + 3) + 2;
   }
   ```
   求 `func(10)`。  
2. 指出这段递归的出口条件是什么。

### 备课提醒
- 初赛出现时一般不会太难，重点是“会顺着算”。
- 建议用树状图或“展开链”讲解。

---

## 15. 二叉树、完全二叉树等拓展常识题

### 知识点
- 完全二叉树的高度
- 节点数与层数关系
- 基础数据结构概念辨析

### 典型问题
- 高度为 1 的完全二叉树，含 62 个节点时高度是多少？
- 完全二叉树的层数如何估计？

### 例题
1. 若根节点高度记为 1，含 62 个节点的完全二叉树高度是多少？

### 备课提醒
- 小学组出现频率不如前面几类高，但近年的题中已有渗透。
- 可作为拔高拓展，不必讲得过深。

# 三、最值得重点刷的高频题型

1. **标识符 / 关键字 / 注释**
2. **数据类型与常量**
3. **运算符与表达式**
4. **逻辑判断与 if-else**
5. **循环与程序执行过程**
6. **数组下标与遍历**
7. **字符串遍历与统计**
8. **读程序写结果**

其中最重要的是：
- **整数除法、浮点运算、取模 `%`**
- **逻辑表达式真假判断**
- **数组越界与下标从 0 开始**
- **循环边界**
- **逐步模拟程序运行**

# 四、适合课堂整理成专题训练的“典型问题”清单

## 专题 1：哪些写法是合法的？
- 哪个是合法标识符？
- 哪个是关键字？
- 哪个是正确注释？
- 哪段代码语法正确？

## 专题 2：结果到底是多少？
- `14 / 3 * 3`
- `15 / 2` 与 `15 / 2.0`
- `a *= 3`
- `a %= 4`
- `'9' * 3`

## 专题 3：条件判断怎么写？
- 判断两个数都为 0
- 判断两个数都不为 0
- 判断某数是否为偶数
- `if` 与 `if-else` 的结构辨析

## 专题 4：循环边界最容易错在哪里？
- 从 `0` 开始还是从 `1` 开始
- 是 `< n` 还是 `<= n - 1`
- 数组是否越界
- 循环变量什么时候更新

## 专题 5：数组和字符串怎么遍历？
- 数组元素批量修改
- 二维数组默认补零
- 最长连续相同字符统计
- 比较 `s[i]` 和 `s[i - 1]`

## 专题 6：读程序写输出
- 复合赋值
- 连续 `if`
- 分支 + 循环
- 递归展开

# 五、备课建议

## 1. 先讲“概念题”，再讲“程序阅读题”
建议顺序：
- 语法规则
- 运算规则
- 分支循环
- 数组字符串
- 读程序写结果

## 2. 每个知识点都配“易错点”
例如：
- `'3'` 不是整数，是字符
- `a == b == 0` 是错的常见写法
- `%` 不能算小数
- 数组下标从 0 开始
- `do...while` 至少执行一次

## 3. 多用“补空题”训练
初赛很喜欢：
- 补判断条件
- 补循环变量
- 补数组下标
- 补更新语句

## 4. 训练学生“手动模拟程序”
可让学生画三列表：
- 当前步骤
- 变量值
- 输出内容

这对初赛提升非常明显。

# 七、总结

从你这个资料目录来看，**初赛核心并不在“难算法”，而在“基础语法 + 程序阅读 + 简单思维”**。  
如果备课时间有限，建议把精力主要放在：

- **运算表达式**
- **逻辑判断**
- **循环与数组**
- **字符串统计**
- **读程序写结果**

这几块最容易连续出题，也最容易拉开分数。
