### 题 3：二元属性相似度计算

**已知数据：** 10个二元属性。

- $X_1$: `1 0 1 1 1 1 1 0 1 1`
    
- $X_2$: `1 1 0 0 1 0 0 1 1 0`
    
- $X_3$: `0 1 1 0 1 1 1 0 0 1`
    

我们首先定义用于二元比较的频率：

- $f_{11}$：两个对象都取 1 的属性个数
    
- $f_{10}$：第一个对象取 1、第二个对象取 0 的属性个数
    
- $f_{01}$：第一个对象取 0、第二个对象取 1 的属性个数
    
- $f_{00}$：两个对象都取 0 的属性个数
    
- 总属性数 $N = f_{11} + f_{10} + f_{01} + f_{00} = 10$
    

相关公式：

- **简单匹配系数 (SMC):** $s_{smc} = \frac{f_{11} + f_{00}}{N}$
    
- **Jaccard 系数:** $s_{jc} = \frac{f_{11}}{f_{11} + f_{10} + f_{01}}$
    
- **Rao 系数:** $s_{rc} = \frac{f_{11}}{N}$
    

**1. 计算 $(X_1, X_2)$ 的相似度**

对比 $X_1$ 和 $X_2$ 的每一位：

- $f_{11}$ (均为1): 第1, 5, 9位，共 **3** 个
    
- $f_{10}$ ($X_1$为1,$X_2$为0): 第3, 4, 6, 7, 10位，共 **5** 个
    
- $f_{01}$ ($X_1$为0,$X_2$为1): 第2, 8位，共 **2** 个
    
- $f_{00}$ (均为0): **0** 个
    
- **答案：**
    
    - $s_{smc}(X_1, X_2) = \frac{3 + 0}{10} = 0.3$
        
    - $s_{jc}(X_1, X_2) = \frac{3}{3 + 5 + 2} = 0.3$
        
    - $s_{rc}(X_1, X_2) = \frac{3}{10} = 0.3$
        

**2. 计算 $(X_2, X_3)$ 的相似度**

对比 $X_2$ 和 $X_3$ 的每一位：

- $f_{11}$ (均为1): 第2, 5位，共 **2** 个
    
- $f_{10}$ ($X_2$为1,$X_3$为0): 第1, 8, 9位，共 **3** 个
    
- $f_{01}$ ($X_2$为0,$X_3$为1): 第3, 6, 7, 10位，共 **4** 个
    
- $f_{00}$ (均为0): 第4位，共 **1** 个
    
- **答案：**
    
    - $s_{smc}(X_2, X_3) = \frac{2 + 1}{10} = 0.3$
        
    - $s_{jc}(X_2, X_3) = \frac{2}{2 + 3 + 4} = \frac{2}{9} \approx 0.222$
        
    - $s_{rc}(X_2, X_3) = \frac{2}{10} = 0.2$

### 题 4：分类属性相似度计算

**公式：** 对于 $p$ 个分类属性，相似度 $s(X_i, X_j) = \frac{m}{p}$，其中 $m$ 是匹配的属性个数，$p=5$。

- $X_1$: (红, 单身, 女, B, 工人)
    
- $X_2$: (白, 离异, 男, AB, 工人)
    
- $X_3$: (蓝, 单身, 男, B, 教师)
    

**1. 计算 $s(X_1, X_2)$**

- 匹配的属性：仅有“职业” (工人) 相同，即 $m = 1$。
    
- $s(X_1, X_2) = \frac{1}{5} = 0.2$
    

**2. 计算 $s(X_1, X_3)$**

- 匹配的属性：“婚姻状况” (单身) 和 “血型” (B) 相同，即 $m = 2$。
    
- $s(X_1, X_3) = \frac{2}{5} = 0.4$
    

**3. 计算 $s(X_2, X_3)$**

- 匹配的属性：“性别” (男) 相同，即 $m = 1$。
    
- $s(X_2, X_3) = \frac{1}{5} = 0.2$

### 题 5：序数属性相似度计算

处理序数属性时，通常先将第 $f$ 个属性的值映射为其排位 $r_{if} \in \{1, \dots, M_f\}$，然后计算对象 $i$ 和 $j$ 在该属性上的距离 $d_f(i, j) = \frac{|r_{if} - r_{jf}|}{M_f - 1}$。综合相似度为 $s(X_i, X_j) = 1 - \frac{1}{p}\sum d_f(i,j)$。

**排位映射：**

- **成绩 ($M_1=5$):** 优(1), 良(2), 中(3), 及格(4), 不及格(5) _(注：升序或降序排位不影响最终差值绝对值)_
    
- **奖学金 ($M_2=3$):** 甲(1), 乙(2), 丙(3)
    
- **月消费 ($M_3=3$):** 高(1), 中(2), 低(3)
    

**对象数值化：**

- $X_1$: (良, 甲, 高) $\rightarrow (2, 1, 1)$
    
- $X_2$: (优, 甲, 中) $\rightarrow (1, 1, 2)$
    
- $X_3$: (中, 丙, 高) $\rightarrow (3, 3, 1)$
    

**1. $s(X_1, X_2)$**

- 距离求和：$d = \frac{|2-1|}{5-1} + \frac{|1-1|}{3-1} + \frac{|1-2|}{3-1} = \frac{1}{4} + 0 + \frac{1}{2} = 0.75$
    
- 平均距离 = $\frac{0.75}{3} = 0.25$
    
- $s(X_1, X_2) = 1 - 0.25 = 0.75$
    

**2. $s(X_1, X_3)$**

- 距离求和：$d = \frac{|2-3|}{4} + \frac{|1-3|}{2} + \frac{|1-1|}{2} = \frac{1}{4} + 1 + 0 = 1.25$
    
- 平均距离 = $\frac{1.25}{3} \approx 0.417$
    
- $s(X_1, X_3) = 1 - 0.417 = 0.583$
    

**3. $s(X_2, X_3)$**

- 距离求和：$d = \frac{|1-3|}{4} + \frac{|1-3|}{2} + \frac{|2-1|}{2} = \frac{2}{4} + 1 + \frac{1}{2} = 2.0$
    
- 平均距离 = $\frac{2.0}{3} \approx 0.667$
    
- $s(X_2, X_3) = 1 - 0.667 = 0.333$

### 题 7：混合属性相异度矩阵计算

混合属性的相异度公式为：

$$d(i,j) = \frac{\sum_{f=1}^p \delta_{ij}^{(f)} d_{ij}^{(f)}}{\sum_{f=1}^p \delta_{ij}^{(f)}}$$

如果对象 $i$ 或 $j$ 的属性 $f$ 为空缺值 (Null)，则指示符 $\delta_{ij}^{(f)} = 0$，否则为 1。

**属性类型分析：**

1. **性别** (对称二元)：相同距离为 0，不同为 1。
    
2. **婚姻状况** (名义)：相同为 0，不同为 1。
    
3. **学位** (名义)：相同为 0，不同为 1。(由于存在“其他”，作为名义变量处理最稳妥)。
    
4. **当月消费额** (数值)：距离计算为 $\frac{|x_i - x_j|}{\max - \min}$。其中非空数据最大值为 3670 ($X_4$)，最小值为 1025 ($X_5$)，极差为 $3670 - 1025 = 2645$。
    

我们计算各点对距离 $d(i,j)$ （保留3位小数）：

- **$d(X_2, X_1)$:** 婚姻、消费含Null ($\delta=0$)。性(男=男,0)、学(硕士$\neq$其他,1)。
    
    $d = \frac{0 + 1}{2} = 0.500$
    
- **$d(X_3, X_1)$:** 性(0), 婚(离异$\neq$已婚,1), 学(1), 消($\frac{|3586-1230|}{2645} \approx 0.891$)。
    
    $d = \frac{0 + 1 + 1 + 0.891}{4} \approx 0.723$
    
- **$d(X_3, X_2)$:** 婚、消含Null。性(0), 学(1)。
    
    $d = \frac{0 + 1}{2} = 0.500$
    
- **$d(X_4, X_1)$:** 性(女$\neq$男,1), 婚(1), 学(1), 消($\frac{|3670-1230|}{2645} \approx 0.922$)。
    
    $d = \frac{1 + 1 + 1 + 0.922}{4} \approx 0.981$
    
- **$d(X_4, X_2)$:** 婚、消含Null。性(1), 学(硕士=硕士,0)。
    
    $d = \frac{1 + 0}{2} = 0.500$
    
- **$d(X_4, X_3)$:** 性(1), 婚(1), 学(1), 消($\frac{|3670-3586|}{2645} \approx 0.032$)。
    
    $d = \frac{1 + 1 + 1 + 0.032}{4} \approx 0.758$
    
- **$d(X_5, X_1)$:** 性(0), 婚(1), 学(1), 消($\frac{|1025-1230|}{2645} \approx 0.078$)。
    
    $d = \frac{0 + 1 + 1 + 0.078}{4} \approx 0.519$
    
- **$d(X_5, X_2)$:** 婚、消含Null。性(0), 学(1)。
    
    $d = \frac{0 + 1}{2} = 0.500$
    
- **$d(X_5, X_3)$:** 性(0), 婚(1), 学(1), 消($\frac{|1025-3586|}{2645} \approx 0.968$)。
    
    $d = \frac{0 + 1 + 1 + 0.968}{4} \approx 0.742$
    
- **$d(X_5, X_4)$:** 性(1), 婚(单身=单身,0), 学(1), 消($\frac{|1025-3670|}{2645} = 1.0$)。
    
    $d = \frac{1 + 0 + 1 + 1}{4} = 0.750$
    
- **$d(X_6, X_1)$:** 学含Null。性(1), 婚(1), 消($\frac{|2890-1230|}{2645} \approx 0.628$)。
    
    $d = \frac{1 + 1 + 0.628}{3} \approx 0.876$
    
- **$d(X_6, X_2)$:** 婚、学、消均含Null！仅对比性别。性(女$\neq$男,1)。
    
    $d = \frac{1}{1} = 1.000$
    
- **$d(X_6, X_3)$:** 学含Null。性(1), 婚(1), 消($\frac{|2890-3586|}{2645} \approx 0.263$)。
    
    $d = \frac{1 + 1 + 0.263}{3} \approx 0.754$
    
- **$d(X_6, X_4)$:** 学含Null。性(女=女,0), 婚(1), 消($\frac{|2890-3670|}{2645} \approx 0.295$)。
    
    $d = \frac{0 + 1 + 0.295}{3} \approx 0.432$
    
- **$d(X_6, X_5)$:** 学含Null。性(1), 婚(1), 消($\frac{|2890-1025|}{2645} \approx 0.705$)。
    
    $d = \frac{1 + 1 + 0.705}{3} \approx 0.902$
