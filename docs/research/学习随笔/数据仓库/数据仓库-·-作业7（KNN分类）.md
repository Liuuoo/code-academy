# 数据仓库 · 作业7（KNN分类）


### 第6题：$k$-最近邻分类算法 (KNN)

**目标**：预测后天 $X_H = (\text{雨}, \text{高}, \text{小}, \text{无})$ 是否适合打球 ($k=3$)。

1. **计算距离**：
    
    根据题目定义的相异度度量公式 $d(X,Y) = \sum_{j=1}^4 \delta(x_j, y_j)$，即计算对应属性值不相同的个数。
    
    将 $X_H$ 与样本集 $S$ 中的 14 个样本依次比较：
    
    - $d(X_H, X_1) = 2$ (天气、湿度不同)
        
    - $d(X_H, X_2) = 2$ (天气、湿度不同)
        
    - $d(X_H, X_3) = 2$ (天气、湿度不同)
        
    - $d(X_H, X_4) = 2$ (温度、湿度不同)
        
    - $d(X_H, X_5) = 1$ (温度不同) —— **距离最近**
        
    - $d(X_H, X_6) = 2$ (温度、风力不同)
        
    - $d(X_H, X_7) = 3$ (天气、温度、风力不同)
        
    - $d(X_H, X_8) = 3$ (天气、温度、湿度不同)
        
    - $d(X_H, X_9) = 2$ (天气、温度不同)
        
    - $d(X_H, X_{10}) = 1$ (温度不同) —— **距离最近**
        
    - $d(X_H, X_{11}) = 3$ (天气、温度、风力不同)
        
    - $d(X_H, X_{12}) = 4$ (四项均不同)
        
    - $d(X_H, X_{13}) = 1$ (天气不同) —— **距离最近**
        
    - $d(X_H, X_{14}) = 3$ (温度、湿度、风力不同)
        
2. **确定 $k$ 个最近邻**：
    
    当 $k=3$ 时，距离最近的 3 个样本为 $X_5, X_{10}, X_{13}$，它们的距离均为 1。
    
3. **多数表决进行预测**：
    
    - $X_5$ 的类别为：**是**
        
    - $X_{10}$ 的类别为：**是**
        
    - $X_{13}$ 的类别为：**是**
        
    
    3个最近邻中，类别为“是”的比例占绝对多数（3票对0票）。因此，预测后天 **是** 适合打球。

### 第7题：ID3 算法生成决策树与分类规则

_(注：题目描述提及8个数据对象，但表9-21中仅展示了 $X_1$ 至 $X_6$ 共6个样本。以下计算严格基于表中可见的这6个样本进行求解。)_

**数据集分析**：

6个样本中，类别（是否卵生）为 1 的有 4 个（$X_1, X_2, X_3, X_4$），为 0 的有 2 个（$X_5, X_6$）。

初始信息熵：$Info(S) = -\frac{4}{6}\log_2(\frac{4}{6}) - \frac{2}{6}\log_2(\frac{2}{6}) \approx 0.918$

1. **计算各属性的信息增益**：
    
    - **是否温血 (A)**：取值为1(5个样本,其中3个1和2个0)，取值为0(1个样本,其中1个1)。
        
        $Info_A(S) = \frac{5}{6}[-\frac{3}{5}\log_2(\frac{3}{5})-\frac{2}{5}\log_2(\frac{2}{5})] + \frac{1}{6}[0] \approx 0.809$
        
        $Gain(A) = 0.918 - 0.809 = 0.109$
        
    - **有无羽毛 (B)**：取值为1(3个样本: $X_1,X_3,X_4$，全为1类)，取值为0(3个样本: $X_2,X_5,X_6$，包含1个1类和2个0类)。
        
        $Info_B(S) = \frac{3}{6}[0] + \frac{3}{6}[-\frac{1}{3}\log_2(\frac{1}{3})-\frac{2}{3}\log_2(\frac{2}{3})] = 0.5 \times 0.918 = 0.459$
        
        $Gain(B) = 0.918 - 0.459 = 0.459$
        
    - **有无毛皮 (C)**：取值为1(1个样本,为0类)，取值为0(5个样本，其中4个1类和1个0类)。
        
        $Info_C(S) = \frac{1}{6}[0] + \frac{5}{6}[-\frac{4}{5}\log_2(\frac{4}{5})-\frac{1}{5}\log_2(\frac{1}{5})] \approx 0.602$
        
        $Gain(C) = 0.918 - 0.602 = 0.316$
        
    - **会否游泳 (D)**：取值为1(2个样本,1个1类1个0类)，取值为0(4个样本,3个1类1个0类)。
        
        $Info_D(S) = \frac{2}{6}[1.0] + \frac{4}{6}[0.811] \approx 0.874$
        
        $Gain(D) = 0.918 - 0.874 = 0.044$
        
2. **构建决策树**：
    
    - **根节点**：“有无羽毛”的增益最大（0.459），作为根节点。
        
    - 分支 **有无羽毛=1**：包含样本 $X_1, X_3, X_4$，全部为卵生（类别1），生成叶子节点：**卵生(1)**。
        
    - 分支 **有无羽毛=0**：包含样本 $X_2, X_5, X_6$。在这个子集中继续计算信息增益。
        
        发现属性“是否温血”能完美区分剩余样本：
        
        - 当“是否温血”=0时，仅有 $X_2$（卵生，类别1）。
            
        - 当“是否温血”=1时，有 $X_5, X_6$（非卵生，类别0）。
            
3. **最终生成的分类规则**：
    
    - **规则1**：如果 有无羽毛=1，则 该动物为 卵生(1)。
        
    - **规则2**：如果 有无羽毛=0 且 是否温血=0，则 该动物为 卵生(1)。
        
    - **规则3**：如果 有无羽毛=0 且 是否温血=1，则 该动物为 非卵生(0)。

### 第8题：C4.5 算法构造决策树

**数据离散化处理**：

按题目规则将“湿度”连续值离散化：

- $<75$ 为**低**：$X_6, X_7, X_9, X_{11}$
    
- $75 \sim 85$ 为**中**：$X_3, X_4, X_5, X_8, X_{10}, X_{12}, X_{13}, X_{14}$
    
- $>85$ 为**高**：$X_1, X_2$
    

**C4.5 算法特性**：

C4.5 使用**信息增益率 (Gain Ratio)** 来选择划分属性，以克服 ID3 偏向选择取值较多的属性的缺点。

1. **选择根节点**：
    
    整体数据集信息熵 $Info(S) = -\frac{9}{14}\log_2(\frac{9}{14}) - \frac{5}{14}\log_2(\frac{5}{14}) \approx 0.940$
    
    计算“天气”与离散化后“湿度”的增益率：
    
    - **天气**的增益：$Gain \approx 0.247$，分裂信息 $SplitInfo \approx 1.577$，**信息增益率 $\approx 0.156$**
        
    - **湿度**的增益：$Gain \approx 0.246$，分裂信息 $SplitInfo \approx 1.378$，**信息增益率 $\approx 0.178$**
        
    
    由于湿度的信息增益率最高，选择 **湿度** 作为根节点。
    
2. **树的构造**：
    
    - **湿度 = 高**：子集包含 $X_1, X_2$，均为“否”，生成叶子节点 **否**。
        
    - **湿度 = 低**：子集包含 $X_6, X_7, X_9, X_{11}$。在此子集上，用“天气”进行划分可得纯净子集（雨->否，云/晴->是）。
        
    - **湿度 = 中**：子集包含 8 个样本。在此子集上计算剩余属性的增益率，发现“天气”增益率最高，作为内部节点。
        
        - 天气 = 晴 -> 叶子节点 **否** ($X_8$)
            
        - 天气 = 云 -> 叶子节点 **是** ($X_3, X_{12}, X_{13}$)
            
        - 天气 = 雨 -> 还有正负样本混杂，继续使用“风力”划分：有风->否($X_{14}$)，无风->是($X_4, X_5, X_{10}$)。
            

**最终的 C4.5 决策树结构**：

Plaintext

```
湿度
 ├── 高 -> 否
 ├── 低 -> 天气
 │          ├── 晴 -> 是
 │          ├── 云 -> 是
 │          └── 雨 -> 否
 └── 中 -> 天气
            ├── 晴 -> 否
            ├── 云 -> 是
            └── 雨 -> 风力
                       ├── 有 -> 否
                       └── 无 -> 是
```

### 第9题：朴素贝叶斯分类预测

**待预测新顾客 $X$** = (学生=是，年龄$\le$30岁，收入=中等，信用=一般)

_(注：根据表9-23图片原始字形读取，$X_{10}$与$X_{11}$的年龄字段首字符明显为"$\ge$"，即$\ge$41岁。以下概率统计均基于图片实际可见内容严格统计)_

1. **计算先验概率**：
    
    总样本数 = 14
    
    - $P(\text{买电脑=是}) = 9 / 14$
        
    - $P(\text{买电脑=否}) = 5 / 14$
        
2. **计算在“买电脑=是”条件下的各属性似然概率** (共9个样本)：
    
    - $P(\text{学生=是} | \text{是}) = 6 / 9$ (包含 $X_5, X_7, X_9, X_{10}, X_{11}, X_{13}$)
        
    - $P(\text{年龄}\le\text{30岁} | \text{是}) = 1 / 9$ (仅包含 $X_9$)
        
    - $P(\text{收入=中} | \text{是}) = 4 / 9$ (包含 $X_4, X_{10}, X_{11}, X_{12}$)
        
    - $P(\text{信用=一般} | \text{是}) = 6 / 9$ (包含 $X_3, X_4, X_5, X_9, X_{10}, X_{13}$)
        
    
    **$P(X | \text{是}) = \frac{6}{9} \times \frac{1}{9} \times \frac{4}{9} \times \frac{6}{9} = \frac{16}{729} \approx 0.0219$**
    
3. **计算在“买电脑=否”条件下的各属性似然概率** (共5个样本)：
    
    - $P(\text{学生=是} | \text{否}) = 1 / 5$ (仅包含 $X_6$)
        
    - $P(\text{年龄}\le\text{30岁} | \text{否}) = 3 / 5$ (包含 $X_1, X_2, X_8$)
        
    - $P(\text{收入=中} | \text{否}) = 2 / 5$ (包含 $X_8, X_{14}$)
        
    - $P(\text{信用=一般} | \text{否}) = 2 / 5$ (包含 $X_1, X_8$)
        
    
    **$P(X | \text{否}) = \frac{1}{5} \times \frac{3}{5} \times \frac{2}{5} \times \frac{2}{5} = \frac{12}{625} \approx 0.0192$**
    
4. **计算后验概率并比较**：
    
    - 买电脑的相对概率：$P(X|\text{是}) \times P(\text{是}) = (\frac{16}{729}) \times (\frac{9}{14}) = \frac{8}{567} \approx \mathbf{0.0141}$
        
    - 不买电脑的相对概率：$P(X|\text{否}) \times P(\text{否}) = (\frac{12}{625}) \times (\frac{5}{14}) = \frac{6}{875} \approx \mathbf{0.00686}$
        

**结论**：因为 $0.0141 > 0.00686$，所以朴素贝叶斯分类器预测该新顾客 $X$ **会买电脑（预测结果为“是”）**。
