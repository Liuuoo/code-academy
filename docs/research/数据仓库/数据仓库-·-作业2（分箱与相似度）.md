# 数据仓库 · 作业2（分箱与相似度）


### 第 6 题

**题目：** 设数据集 $A=\{1,1,3,3,4,5,5,5,6,7,7,7,9,10,12,15\}$，请用等深分箱法将其分成 $k=4$ 个箱子。

**解答：**

等深分箱法（又称等频分箱法）是指每个箱子中包含的数据个数相同。

1. 数据集 $A$ 已经从小到大排序，数据总个数 $N = 16$。
    
2. 需要分成 $k = 4$ 个箱子，因此每个箱子包含的数据个数为：$N / k = 16 / 4 = 4$ 个。
    
3. 按顺序将数据放入箱子中：
    
    - **箱子 1：** $\{1, 1, 3, 3\}$
        
    - **箱子 2：** $\{4, 5, 5, 5\}$
        
    - **箱子 3：** $\{6, 7, 7, 7\}$
        
    - **箱子 4：** $\{9, 10, 12, 15\}$

### 第 7 题

**题目：** 设数据集 $A=\{1,1,3,3,4,5,5,5,6,7,7,7,9,10,12,15\}$，请用等宽分箱法将其分成 $k=4$ 个箱子。

**解答：**

等宽分箱法是指每个箱子所覆盖的数据范围（区间宽度）相同。

1. 数据集的最大值 $Max = 15$，最小值 $Min = 1$。
    
2. 划分为 $k = 4$ 个箱子，区间宽度 $W$ 计算如下：
    
    $$W = \frac{Max - Min}{k} = \frac{15 - 1}{4} = \frac{14}{4} = 3.5$$
    
3. 划分区间边界（通常左闭右开，最后一个区间包含最大值）：
    
    - 区间 1：$[1, 1+3.5) = [1, 4.5)$
        
    - 区间 2：$[4.5, 4.5+3.5) = [4.5, 8)$
        
    - 区间 3：$[8, 8+3.5) = [8, 11.5)$
        
    - 区间 4：$[11.5, 15]$
        
4. 将数据分配到对应的箱子中：
    
    - **箱子 1 $[1, 4.5)$：** $\{1, 1, 3, 3, 4\}$
        
    - **箱子 2 $[4.5, 8)$：** $\{5, 5, 5, 6, 7, 7, 7\}$
        
    - **箱子 3 $[8, 11.5)$：** $\{9, 10\}$
        
    - **箱子 4 $[11.5, 15]$：** $\{12, 15\}$

### 第 12 题

**题目：** 设有实数区间 $[15,30]$，请用最小-最大规范化方法将 $x=22 \in [15,30]$ 变换为 $x' \in [50,95]$。

**解答：**

最小-最大规范化公式为：

$$x' = \frac{x - \min}{\max - \min} \times (\text{new\_max} - \text{new\_min}) + \text{new\_min}$$

已知：$x = 22$，$\min = 15$，$\max = 30$，$\text{new\_min} = 50$，$\text{new\_max} = 95$。代入公式：

$$x' = \frac{22 - 15}{30 - 15} \times (95 - 50) + 50$$

$$x' = \frac{7}{15} \times 45 + 50$$

$$x' = 7 \times 3 + 50$$

$$x' = 21 + 50 = 71$$

**结果：** 变换后的值 $x' = 71$。

### 第 13 题

**题目：** 设有数据集 $A=\{1,5,6,7,9,10,12,15\}$，试用零-均值规范化方法对其进行规范化。

**解答：**

零-均值规范化（Z-score 规范化）公式为：

$$x' = \frac{x - \mu}{\sigma}$$

其中 $\mu$ 为数据集的均值，$\sigma$ 为标准差。数据总数 $N = 8$。

1. **计算均值 $\mu$：**
    
    $$\mu = \frac{1+5+6+7+9+10+12+15}{8} = \frac{65}{8} = 8.125$$
    
2. **计算标准差 $\sigma$（此处使用总体标准差）：**
    
    $$\sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{N}}$$
    
    方差计算：
    
    $\frac{(-7.125)^2 + (-3.125)^2 + (-2.125)^2 + (-1.125)^2 + (0.875)^2 + (1.875)^2 + (3.875)^2 + (6.875)^2}{8}$
    
    $= \frac{50.766 + 9.766 + 4.516 + 1.266 + 0.766 + 3.516 + 15.016 + 47.266}{8} = \frac{132.875}{8} = 16.609375$
    
    $\sigma \approx \sqrt{16.609} \approx 4.075$
    
3. **计算规范化后的值：**
    
    - $x'_1 = (1 - 8.125) / 4.075 \approx -1.748$
        
    - $x'_2 = (5 - 8.125) / 4.075 \approx -0.767$
        
    - $x'_3 = (6 - 8.125) / 4.075 \approx -0.521$
        
    - $x'_4 = (7 - 8.125) / 4.075 \approx -0.276$
        
    - $x'_5 = (9 - 8.125) / 4.075 \approx 0.215$
        
    - $x'_6 = (10 - 8.125) / 4.075 \approx 0.460$
        
    - $x'_7 = (12 - 8.125) / 4.075 \approx 0.951$
        
    - $x'_8 = (15 - 8.125) / 4.075 \approx 1.687$
        

_(注：如果课本要求使用样本标准差 $N-1$，则 $\sigma \approx 4.357$，计算出的对应值会略有不同。通常计算题默认采用总体标准差。)_

### 第 14 题

**题目：** 试述多维数据模型的定义。

**解答：**

多维数据模型是为了满足用户从多角度、多层次进行数据查询和分析而建立的数据模型。它通常将数据组织成“数据立方体（Data Cube）”的形式，主要包含两个核心概念：**维度（Dimension）**和**事实（Fact）**。维度是观察数据的角度（如时间、地点、产品等），事实是具体要度量的指标数据（如销售额、数量等）。该模型是数据仓库和联机分析处理（OLAP）的基础。

### 第 16 题

**题目：** 数据仓库有哪几种逻辑模型？

**解答：**

在关系型数据库中实现多维数据结构，数据仓库主要有以下三种逻辑模型：

1. **星型模型（Star Schema）：** 最常见的模型，中心是一个事实表，周围环绕着多个单一的维度表。
    
2. **雪花模型（Snowflake Schema）：** 星型模型的扩展，对维度表进一步规范化，将其拆分成多个相关的表，形状类似雪花。
    
3. **事实星座模型（Fact Constellation Schema）/ 星系模型：** 包含多个事实表，且这些事实表之间共享某些维度表。

### 第 17 题

**题目：** 位图索引是数据仓库的什么模型？

**解答：**

位图索引属于数据仓库的**物理数据模型**（物理层面的设计）。

逻辑模型（如星型模型）关注的是数据的组织结构和业务逻辑，而物理数据模型关注的是数据在数据库底层的实际存储、访问方式和性能优化。位图索引是一种专门用于优化复杂多维查询性能的数据库索引技术。
