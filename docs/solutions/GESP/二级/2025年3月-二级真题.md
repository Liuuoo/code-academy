## 二、 单选题精讲 (每题2分)

### 1. 科技时事与常识

- **题目内容**：关于DeepSeek与《哪吒2》的描述。
    
- **解析**：
    
    - DeepSeek 是近期热门的 **人工智能(AI)大模型**。
        
    - 《哪吒2》是 **电影**。
        
    - **A/C错误**：电影不是操作系统或软件。
        
    - **B错误**：DeepSeek不是钻探软件。
        
    - **D正确**：生成式AI（如DeepSeek）具备根据场景生成文本（剧本）的能力。
        
- **知识点**：计算机科学常识。
    

### 2. 流程图符号

- **题目内容**：输出语句应该用哪种图形框？
    
- **解析**：
    
    - **平行四边形 (Parallelogram)**：用于 **输入/输出 (Input/Output)**。
        
    - 矩形：用于处理/计算。
        
    - 菱形：用于判断。
        
    - 圆角矩形/椭圆：用于开始/结束。
        
- **答案**：C (平行四边形框) 1
    

### 3. C++ 比较与赋值的区别

- **题目内容**：`int a=3, b=4; a==b; b==a; cout << a << b;`
    
- **解析**：
    
    - **陷阱**：`==` 是 **比较运算符**，不是赋值运算符。
        
    - 语句 `a == b;` 只是比较两数是否相等，产生一个 `false` 结果，但**不会改变** `a` 或 `b` 的值。
        
    - 因此，`a` 仍为 3，`b` 仍为 4。
        
- **答案**：A (34) 2
    

### 4. 周期问题与取余 (Mod)

- **题目内容**：红绿蓝球循环 (5红, 3绿, 2蓝)，总周期 $T=10$。求编号 $N$ 的颜色。
    
- **解析**：
    
    - 计算位置：`remainder = N % 10`。
        
    - **范围映射**：
        
        - 余数 1~5: 红色。
            
        - 余数 6~8: 绿色。
            
        - 余数 9, 0: 蓝色 (因为 $10 \% 10 = 0$，第10个是蓝色)。
            
    - **选项分析**：
        
        - **A (正确)**：原代码 `(remainder == 9) || (remainder == 0)` 涵盖了剩下的所有情况，直接用 `else` 效果完全相同。
            
        - B (错误)：`remainder <= 5` 包含了 0，而 0 应该是蓝色，原代码 1-5 才是红色。
            
        - C (错误)：C++中 `6 <= remainder <= 8` 会先算 `6 <= remainder` 得到 0或1，再算 `0/1 <= 8`，逻辑恒真，这是经典的语法错误。
            
        - D (错误)：求位置必须用取余 `%`，不能用除法 `/`。
            
- **答案**：A 3
    

### 5. 循环与条件计算

- **题目内容**：`for` 循环 `i` 从 0 到 9。`if (i % 3)` tnt+=1 `else` tnt+=2。
    
- **解析**：
    
    - `i % 3` 的结果：
        
        - 当 `i = 0, 3, 6, 9`: `i % 3 == 0` (False)，执行 `else` (tnt+=2)。共4个数，加 $4 \times 2 = 8$。
            
        - 当 `i = 1, 2, 4, 5, 7, 8`: `i % 3 != 0` (True)，执行 `if` (tnt+=1)。共6个数，加 $6 \times 1 = 6$。
            
    - **总和**：$8 + 6 = 14$。
        
- **答案**：D (14) 4
    

### 6. 循环中的 Break

- **题目内容**：`for(i=10; i>0; i-=2)` 内部第一句是 `break`。
    
- **解析**：
    
    - 循环初始化 `i=10`。
        
    - 检查条件 `10 > 0` (成立)。
        
    - 进入循环体 -> 执行 `break` -> **立即跳出循环**。
        
    - 此时 `i` 的值没有经过 `i-=2`，仍然是 10。
        
- **答案**：A (10) 5
    

### 7. Continue 的使用

- **题目内容**：`i` 0~9，`if (i%3==0) continue;` 否则输出 `0#`。循环外 `if(i>=10)` 输出 `1#`。
    
- **解析**：
    
    - **被跳过 (continue)**：0, 3, 6, 9 (共4次)。
        
    - **执行输出**：1, 2, 4, 5, 7, 8 (共6次)。所以输出6个 `0#`。
        
    - 循环结束后 `i` 变为 10。
        
    - 循环外判断：`if (10 >= 10)` 成立，输出 `1#`。
        
    - **结果**：6个 `0#` 加上 1个 `1#`。
        
- **答案**：D (0#0#0#0#0#0#1#) 6
    

### 8. 嵌套循环轨迹

- **题目内容**：外层 `i: 0->4`，内层 `j` 从 `i` 递减到 1。
    
- **解析**：
    
    - i=0: 内层不执行。
        
    - i=1: j=1。输出 `1-`。
        
    - i=2: j=2, 1。输出 `2-1-`。
        
    - i=3: j=3, 2, 1。输出 `3-2-1-`。
        
    - i=4: j=4, 3, 2, 1。输出 `4-3-2-1-`。
        
    - 拼接：`1-2-1-3-2-1-4-3-2-1-`。
        
- **答案**：A 7
    

### 9. 逻辑表达式构建

- **题目内容**：能被2整除 **且** 除以7余2。
    
- **解析**：
    
    - 能被2整除（偶数）：`i % 2 == 0` 或 `!(i % 2)`。
        
    - 除以7余2：`i % 7 == 2`。
        
    - **选项检视**：
        
        - A: `(i%2==0) && (i%7==2)` -> 正确。
            
        - B: `(!(i%2))` (偶数真) `&& (i%7==2)` -> 正确。
            
        - C: `(!(i%7))` 表示能被7整除（余数为0），题目要求余2 -> **错误**。
            
        - D: `(i%2 != 1)` (余数不为1即为0) -> 正确。
            
- **答案**：C (不能实现) 8
    

### 10. 数位分离与计数 (填空)

- **题目内容**：统计含有数字3的数的个数。
    
- **解析**：
    
    - **逻辑**：对每个数 `i`，用 `while` 拆分每一位。
        
    - 如果发现某一位 `j % 10 == 3`：
        
        - `cnt` 加 1。
            
        - **关键点**：一个数只要含有一个3就算，不用重复统计。必须停止对当前数 `i` 的检查，跳出 `while` 循环，继续下一个 `i`。
            
        - **空1**：`break;`
            
    - 如果当前位不是3，需要检查下一位：
        
        - **空2**：`j /= 10;` (去掉个位)。
            
- **答案**：B (break; j/=10) 9
    

### 11. 阶乘和的效率

- **题目内容**：比较两种求阶乘和的代码。
    
    - 实现1：`last *= i; sum += last;` (利用前一项结果，O(N))。
        
    - 实现2：双重循环，每次重新算阶乘 (O(N^2))。
        
- **解析**：
    
    - 实现1明显效率更高，因为它避免了重复计算。例如算 $4!$ 时，直接用 $3! \times 4$。
        
- **答案**：C (实现1效率高) 10
    

### 12. 哥德巴赫猜想验证

- **题目内容**：验证偶数分解为两个质数。
    
- **解析**：
    
    - **A**：`expr` 与 `expr == true` 在布尔上下文中等价。正确。
        
    - **B**：`j` 从 2 开始递增，所以找到的第一对 `(j, i-j)` 中，`j` 必然是较小的那个。正确。
        
    - **C**：计算机只能“验证”范围内成立，不能从数学上“证明”无穷的情况。正确。
        
    - **D (错误)**：`break` 是为了找到一组解后就停止，输出该解。如果移到 `if` 外面，循环执行一次无论是否找到都会退出，导致逻辑错误。
        
- **答案**：D 11
    

### 13. 数字三角形

- **题目内容**：输出特定形状的数字阵列。
    
- **解析**：
    
    - **B**：`last += 1` 和 `last = last + 1` 是等价的。正确。
        
    - **A**：`printf` 完全可以输出 `\n`，这是标准用法。错误。
        
    - **C**：`j < i+1` 等同于 `j <= i` (整数)，表示打印 `i` 个数。如果改成 `j < i` 则少打一个。错误。
        
    - **D**：如果 `last=0`，第一个数会是0，不符合样例的1。错误。
        
- **答案**：B 12
    

### 14. 随机数范围

- **题目内容**：生成 1 到 10 的随机整数。
    
- **解析**：
    
    - `rand() % N` 生成 `0` 到 `N-1`。
        
    - `rand() % 10` 生成 `0` 到 `9`。
        
    - 整体 `+ 1` -> `1` 到 `10`。
        
- **答案**：C (rand()%10 + 1) 13
    

### 15. 浮点数比较

- **题目内容**：判断两个 float 变量 a, b 是否相等。
    
- **解析**：
    
    - 浮点数有精度误差，不能直接用 `==`。
        
    - 标准做法是判断两数之差的绝对值是否小于极小值（epsilon）。
        
    - 公式：`abs(a - b) < 1e-6`。
        
- **答案**：C 14

## 三、 判断题精讲 (每题2分)

1. **高级语言原理**：C++/Python 是高级语言，必须编译或解释成机器指令才能被CPU执行。
    
    - **答案**：✅ 正确 15
        
2. **数位运算**：`N % 10` 得到个位。`N/10` 舍去个位，再 `*10` 得到去掉个位并补0的数。`N - (N/10*10)` 确实等于个位。
    
    - **答案**：✅ 正确 16
        
3. **C++ 连续比较的陷阱**：
    
    - 表达式 `10 <= N <= 12`。
        
    - C++ 计算顺序：先算 `(10 <= N)`。假设 `N=12`，结果为 `true` (即 1)。
        
    - 再算 `1 <= 12`。结果为 `true`。
        
    - 虽然逻辑上这通常是代码错误（数学上想表达区间），但题目问的是“输出是否为true”，按C++执行逻辑，结果确实是 `true`。
        
    - **答案**：✅ 正确 17
        
4. **浮点数精度**：`sqrt(N) * sqrt(N) == N`。
    
    - 对于许多整数（如3），`sqrt(3)` 是无限不循环小数，计算机存的是近似值。相乘后可能得到 `2.999999`，不等于 `3`。
        
    - **答案**：❌ 错误 (Source 281 中第4题未打勾)
        
5. **不使用临时变量交换数值**：
    
    - 代码逻辑：`a=2, b=3` -> `a = -1` -> `b = 2` -> `a = 3`。
        
    - 最终 `a=3, b=2`。`a*b = 6`。
        
    - **答案**：✅ 正确 18
        
6. **Loop Continue**：
    
    - `for` 循环中 `continue` 只是跳过循环体剩余语句，**不影响** `i++`。循环正常执行到 `i=10` 结束。
        
    - **答案**：✅ 正确 19
        
7. **Loop Break**：
    
    - `i=1` 进入循环，遇到 `break`，直接跳出。`i` 保持 1。
        
    - **答案**：✅ 正确 20
        
8. **双层循环次数**：
    
    - `i` 从 0 到 4。内部 `j` 从 0 到 `i-1`。
        
    - 次数：0 + 1 + 2 + 3 + 4 = 10 次。
        
    - **答案**：✅ 正确 21
        
9. **求和循环边界**：
    
    - `i=1; i<5` 累加 1,2,3,4。
        
    - `i=0; i<5` 累加 0,1,2,3,4。
        
    - 因为加0不影响结果，所以总和相同。
        
    - **答案**：✅ 正确 22
        
10. **嵌套循环变量冲突**：
    
    - 内层循环使用了和外层循环**同一个变量** `i` 且修改了它。
        
    - 这会破坏外层循环的计数逻辑，通常导致死循环或不可预期的行为，而不是有序输出 0123。
        
    - **答案**：❌ 错误 23

## 四、 总结

1. **取余运算 (%) 是重中之重**：
    
    - 用于判断倍数 (`n % k == 0`)。
        
    - 用于分离数位 (`n % 10` 取个位, `n / 10` 删个位)。
        
    - 用于周期性问题 (红绿灯、星期几)。
        
2. **循环控制要清晰**：
    
    - **break**: 彻底辞职（退出整个循环）。
        
    - **continue**: 请假一天（跳过本次，继续下一次）。
        
    - **多层循环**: 注意变量名不要重复（外层用 `i`，内层用 `j`）。
        
3. **陷阱题型**：
    
    - **比较 vs 赋值**: `==` vs `=`。
        
    - **数学区间**: C++ 不支持 `10 <= n <= 20`，必须写成 `10 <= n && n <= 20`。
        
    - **浮点数**: 永远不要用 `==` 判断 float/double 相等。
