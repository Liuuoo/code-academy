## 二、 单选题精讲 (每题2分)

### 1. 科技常识：诺贝尔物理学奖与AI

- **题目**：2024年诺贝尔物理学奖颁给了计算机科学家 John J. Hopfield 和 Geoffrey E. Hinton 1。
    
- **解析**：
    
    - 这是计算机领域的重磅新闻。Hinton 被称为“深度学习之父”。他们的研究奠定了现代 **人工智能 (Artificial Intelligence)** 的基础 2。
        
    - 虽然是物理学奖，但表彰的是他们在人工神经网络方面的贡献。
        
- **答案**：C (人工智能) 3
    

### 2. 计算机存储单位

- **题目**：存储基本单位 `B` 代表什么？
    
- **解析**：
    
    - **Bit (b)**：比特，最小单位（0或1）。
        
    - **Byte (B)**：字节，基本存储单位。$1 \text{ Byte} = 8 \text{ bits}$ 4。
        
    - 题目问的是 `MB` 中的 `B`，即 **Byte** 5。
        
- **答案**：A 6
    

### 3. 运算符优先级 (易错题)

- **题目**：`cout << (3+3% 3*21)` 7。
    
- **解析**：
    
    - **优先级规则**：`%` (取余) 和 `*` (乘法) 优先级高于 `+` (加法)。同级运算从左到右。
        
    - 第一步：计算 `3 % 3`，结果为 `0`。
        
    - 第二步：计算 `0 * 21`，结果为 `0`。
        
    - 第三步：计算 `3 + 0`，结果为 `3`。
        
    - _注意_：标准答案给的是 **D** (2) 8。这极有可能是题目印刷或OCR识别时漏掉了末尾的运算（例如 `-1`），或者代码并不是标准的 C++ 逻辑。但在 C++ 教学中，必须强调 **取余和乘除先于加减**。
        
- **答案**：D (以考卷答案为准，但需理解正确优先级)
    

### 4. 循环边界

- **题目**：`for (int i=0; i<10; i++)` 输出 `i` 9。
    
- **解析**：
    
    - 初始 `i=0`。
        
    - 条件 `i<10`：意味着 `i` 最大取到 `9`。
        
    - 输出：`0123456789`。
        
- **答案**：B 10
    

### 5. 变量初始化陷阱 (重点)

- **题目**：`int tnt; for(...) tnt += i;` 11。
    
- **解析**：
    
    - **致命错误**：局部变量 `int tnt;` **没有初始化**。
        
    - 在 C++ 中，未初始化的局部变量包含的是内存中的“垃圾值”（随机值）。
        
    - 直接进行 `tnt += i` 计算，结果是不可预测的。
        
- **答案**：D (不确定的值) 12
    

### 6. Break 与 Continue 的配合

- **题目**：`i` 从 1 到 9。`if (i%2) continue; else break;` 13。
    
- **解析**：
    
    - `i=1`: `1%2` 为 1 (True)。执行 `continue` (跳过本次，i变为2)。
        
    - `i=2`: `2%2` 为 0 (False)。执行 `else` 中的 `break` (终止循环)。
        
    - 循环结束，输出 `i`，此时 `i` 为 2。
        
- **答案**：B 14
    

### 7. 循环后的逻辑

- **题目**：`i` 0~9，`i%3` 时 continue，否则打印 `0#`。循环外 `i>=10` 打印 `1#` 15。
    
- **解析**：
    
    - `i` 能被 3 整除时打印 `0#`：0, 3, 6, 9 (共4次)。
        
    - 循环结束后，`i` 变为 10。
        
    - 判断 `if(i>=10)` 成立，打印 `1#`。
        
    - 结果：`0#0#0#0#1#`。
        
- **答案**：C 16
    

### 8. 逻辑表达式构建

- **题目**：能被 7 整除 **且** 不能被 3 整除 17。
    
- **解析**：
    
    - 被 7 整除：`i % 7 == 0` 或 `!(i % 7)`。
        
    - 不能被 3 整除：`i % 3 != 0`。
        
    - **选项 C**：`i % 7 && i % 3` 18。这意味着 `i%7` 不为0（不能被7整除）且 `i%3` 不为0。这与题意相反。
        
- **答案**：C (不能填入的代码) 19
    

### 9. 累加器的使用

- **题目**：求各位数字之和 20。
    
- **解析**：
    
    - 标准写法：`tnt += N % 10` (累加个位)。
        
    - **选项 D**：`tnt = N % 10` 21。这是 **赋值** 操作，会覆盖掉之前计算的和，无法实现累加。
        
- **答案**：D (错误写法) 22
    

### 10. 嵌套循环轨迹

- **题目**：`i` 0~4，`j` 0~`i-1` 23。
    
- **解析**：
    
    - i=0: 不执行。
        
    - i=1: j=0。输出 `0`。
        
    - i=2: j=0,1。输出 `01`。
        
    - i=3: j=0,1,2。输出 `012`。
        
    - i=4: j=0,1,2,3。输出 `0123`。
        
    - 连起来：`0010120123`。
        
- **答案**：A 24
    

### 11. 代码排错 (九九乘法表)

- **题目**：关于换行位置和条件判断 25。
    
- **解析**：
    
    - **选项 A**：建议将外层循环末尾的 `printf("\n")` (L1) 移到内层循环里 (L2) 26。
        
    - 如果移到内层循环里，每输出一个算式就会换一行，这将导致输出变成一列，而不是乘法表的三角形矩阵。所以 A 说法错误。
        
- **答案**：A 27
    

### 12. 阶乘和算法

- **题目**：计算 $1! + 2! + [cite_start]\dots + N!$ 28。
    
- **解析**：
    
    - 核心逻辑：`nowNum` (当前阶乘) 应该每次乘以 `i`。 `tnt` (总和) 应该加上 `nowNum`。
        
    - **选项 D**：`nowNum = nowNum + i` 29。阶乘是乘法，这里用了加法，逻辑完全错误。且 `tnt * nowNum` 是无效语句。
        
- **答案**：D (不能实现) 30
    

### 13. 孪生素数范围

- **题目**：输出 N 和 M 之间 (含) 的孪生素数对 $(i, i+2)$ 31。
    
- **解析**：
    
    - 我们要保证两个数都在 $[N, M]$ 范围内。
        
    - 第二个数是 `i+2`，必须满足 `i+2 <= M`，即 `i <= M - 2`。
        
    - `for` 循环条件是 `i < ?`。
        
    - 若填 `M-1`，则 `i` 最大为 `M-2`，符合要求。
        
- **答案**：B (M-1) 32
    

### 14. 打印金字塔图形

- **题目**：打印高度为 `height` 的星号金字塔 33。
    
- **解析**：
    
    - **空格规律**：第 0 行有 `height-1` 个空格，第 `i` 行有 `height-1-i` 个空格。
        
    - **星号规律**：第 0 行 1 个，第 1 行 3 个... 第 `i` 行 `2*i + 1` 个。
        
    - **选项 D**：空格循环 `height-i-1`，星号循环 `2*i+1`。符合规律。
        
- **答案**：D 34
    

### 15. 表达式计算

- **题目**：`a=10, b=20, c=30`，输出 30。问哪个 **不能** 输出 30 35。
    
- **解析**：
    
    - A: Max 也是 30。
        
    - B: Min(30, 30) 也是 30。
        
    - C: `sqrt(10+20+30)` = $\sqrt{60} \approx 7.74$。**不等于 30**。
        
    - D: Average 是 30。
        
- **答案**：C 36

## 三、 判断题精讲 (每题2分)

1. **文件操作**：Ctrl+C (复制) 和 Ctrl+V (粘贴) 用于建立副本。
    
    - **答案**：✅ 正确 37
        
2. **数位运算**：`N - N/10*10` 利用整数除法舍去小数的特性，确实能得到个位数 (等同于 `%10`)。
    
    - **答案**：✅ 正确 38
        
3. **C++ 连续比较 (经典考点)**：
    
    - 表达式 `10 <= N <= 12` (假设 N=12)。
        
    - C++ 执行顺序：先算 `10 <= 12` $\rightarrow$ `True` (即 1)。
        
    - 再算 `1 <= 12` $\rightarrow$ `True` (即 1)。输出 1。
        
    - **答案**：✅ 正确 39
        
4. **完全平方数判断**：
    
    - `int(sqrt(N)) * int(sqrt(N)) == N`。如果开方取整后再平方等于原数，说明是完全平方数。
        
    - **答案**：✅ 正确 40
        
5. **Printf 格式化**：
    
    - `printf("%%*%%b=%d", ...)`。
        
    - `%%` 在 format string 中输出一个 `%` 字符。
        
    - 所以输出应该是 `%*%b=6`，而不是 `2*3=6`。
        
    - **答案**：❌ 错误 41
        
6. **变量命名规范**：
    
    - `for (int = 0; ...)`。缺少变量名，或者试图用空字符串做变量名，这是语法错误。
        
    - **答案**：✅ 正确 42
        
7. **不可达代码 (Unreachable Code)**：
    
    - `continue` 在 `break` 之前。
        
    - 每次循环都先执行 `continue`，直接跳到下一次循环，`break` 永远不会被执行。
        
    - 循环会正常跑完 10 次，`i` 变为 10。题目说输出 0。
        
    - **答案**：❌ 错误 43
        
8. **双重循环计数**：
    
    - 外层 `i`: 8, 6, 4 (共3次)。
        
    - 内层 `j`: 0~7(8次), 0~5(6次), 0~3(4次)。
        
    - 总次数 $8+6+4=18$。
        
    - **答案**：✅ 正确 44
        
9. **奇数统计**：
    
    - 统计 1~4 中的奇数。无论 `i` 从 0 开始还是 1 开始，`0` 不是奇数，不影响计数结果。
        
    - **答案**：✅ 正确 45
        
10. **For 与 While 等价性**：
    
    - 所有的 `for` 循环都可以改写为 `while` 循环。两段代码逻辑一致。
        
    - **答案**：✅ 正确 46

## 四、 核心知识点总结

1. **优先级铁律**：
    
    - `!` (非) > `* / %` (乘除余) > `+ -` (加减) > `&&` (与) > `||` (或)。
        
    - 遇到 `3+3%3` 这种题，千万别顺着读，先算 `%`。
        
2. **变量一定要初始化**：
    
    - 写 `int sum = 0;` 是好习惯。
        
    - 写 `int sum;` 然后直接 `sum += i;` 是 **Bug**。
        
3. **循环控制三剑客**：
    
    - `break`: 立即跳出循环。
        
    - `continue`: 立即跳过本次，进入下一次。
        
    - `return 0`: 结束整个 `main` 函数。
        
    - _注意_：如果 `continue` 在 `break` 前面无条件执行，`break` 就失效了。
        
4. **数学技巧**：
    
    - **完全平方数**：`x = sqrt(n); if(x*x == n)`
        
    - **数位分离**：`while(n){ digit = n%10; n/=10; }`
        
    - **金字塔公式**：第 $i$ 行 (从0开始) -> 空格 `H-1-i`, 星号 `2*i+1`。
